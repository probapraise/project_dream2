- core question: 키리온은 아버지의 검증망 아래에서 얼마나 오래 정상적인 8세 차남으로 위장할 수 있는가.
- compressed memory:
  - `ep000`에서 칼리온은 질문 구조와 맥락 전환으로 키리온의 정합성을 테스트했다.
  - 키리온은 여덟 살짜리처럼 보이기 위해 답변 속도와 언어 수준을 강제로 낮추는 생존 규칙을 세웠다.
  - `ep001`에서 치유사 검사 자체는 통과했지만, 칼리온은 `흑색 표준식 입문서`를 보내 관찰의 시간을 길게 늘렸다.
  - `ep002`에서 키리온은 안내판으로 첫 설계 성공을 거뒀지만, 그 성공 직후 시험장은 `책을 읽는가`에서 `서고에서 무엇을 먼저 고르는가`로 확장됐다.
- current state:
  - 위협은 즉석 심문에서 장기 관찰로 이동했고, 이제는 선택 구조 전체가 판정 대상이 된다.
  - 작은 성공조차 사고 방식의 출처를 드러내는 진술서가 될 수 있다.
- next trigger:
  - 서고에서 첫 선택을 하는 순간, 혹은 데리온 개입으로 시험장이 뒤틀리는 순간 어떤 리스크가 먼저 폭발하는가.
- load when:
  - 칼리온, 독서, 시험, 위장, 정체 노출 리스크가 중심일 때

## ARC-002 정보 병목 인식과 장기 야망의 현실화
- status: active
- scope: `ep000` -> ongoing
- core question: 주인공의 정보 인프라/커뮤니티 집착은 어떤 현실적 병목과 공동체 온도를 만나 행동 가능한 목표가 되는가.
- compressed memory:
  - `ep000`에서 주인공은 외래 기억 잔향 속 커뮤니티 유지보수 감각과 정보 흐름 설계 습관을 회수했지만, 장기 포부를 크게 선언하기보다 우선 생존으로 수렴했다.
  - `ep001`에서 집사실을 보며, 기록은 존재하지만 접근 인터페이스가 지나치게 좁다는 사실과 사람들끼리 문답을 부풀리며 노는 공동체 온도를 동시에 포착했다.
  - 주인공은 종이 게시판/쓰레드 구조까지 떠올렸지만, 아직 해결 설계로 진입하지 않았고 관찰의 순간에서 멈춰 있었다.
  - `ep002`에서 조악한 안내판 실험이 실제 효용을 증명하며, 장기 야망이 처음으로 작은 현장 개선 경험으로 전환됐다.
- current state:
  - 장기선은 더 이상 추상적 구상만이 아니라, `아이 장난처럼 보이는 인터페이스 개선`이 실제로 먹힌다는 경험적 근거를 얻었다.
  - 다만 효율 개선이 성공할수록 관찰 강도도 함께 올라가므로, 이후의 설계는 위장과 감시 회피를 전제로 해야 한다.
- next trigger:
  - 저택 내부 문서/전달/권한 체계가 추가로 드러날 때
  - 주인공이 다음 인터페이스 병목을 발견하지만, 서고 시험과 가족 감시 때문에 바로 손대지 못할 때
- load when:
  - 정보 인프라, 행정 병목, 커뮤니티 구축 장기선이 중심일 때

## ARC-003 하급 과정 진입과 상급 과정 점프 부트스트랩
- status: active
- scope: `ep000` -> lower academy entry -> main academy jump
- core question: 12세 하급 과정 진입과 15세 상급 과정 점프 사이에 무엇을 먼저 보여줘야 이후 학술원 본편이 가장 강하게 열린다.
- compressed memory:
  - 현재 서사는 넓은 세계를 바로 열지 않고, 렌바렌 저택이라는 좁은 공간에서 감시/돌봄/행정 구조를 먼저 세팅하고 있다.
  - `ep001`에서 남매 모두 언젠가 아카데미에 보내질 것이라는 가문 계획이 제시되어, 외부 무대가 당장 열리지 않는 이유가 구체화됐다.
  - `ep002`에서 저택 내부의 작은 설계 성공과 더 정교해진 가족 감시가 맞물리며, academy 이전 파트를 지탱할 내적 동력이 한 번 더 보강됐다.
  - NC 코어 캐스트 확장과 정식 시뮬레이션은 입학 이후로 미뤄진 상태다.
- current state:
  - 입학 전 파트의 임무는 주인공의 생존 규칙, 가문 분위기, 장기 공론장 욕망의 씨앗, 감시 구도를 고정하는 것이다.
  - 넓은 외부 캐스트를 선제 투입하지 않고, 내부 밀도를 높이는 편이 맞다.
  - 최근 캐논 기준으로는 `작은 승리 -> 더 정교한 감시 -> 다음 선택 압박`의 리듬이 저택 파트를 전진시키는 핵심 엔진이다.
  - 하급 과정 입학식/첫 수업/환경 노출 목표 시점은 잠정적으로 `ep010` 전후다.
  - 그 뒤에는 하급 과정 생활을 길게 끌지 않고, 게시판 설계의 핵심 병목을 파악한 뒤 `3년 후` 상급 과정 진입성취평가와 입학 즉시 전공 선택으로 점프하는 구조가 맞다.
  - 다만 입학 전에는 `칼리온의 최종 확언`까지 가지 않는다. 저택 파트에서는 주인공이 `시험을 일단 넘겼다`는 가벼운 안도만 확보하면 충분하다.
  - `칼리온의 "그럼에도 내 아들이다" 확언`과 주인공의 더 깊은 정체성 수렴은 학술원 진입 후의 가족 이벤트로 미룬다.
- next trigger:
  - 하급 과정 입학 준비라는 외부 일정이 처음 서사 전면에 등장할 때
  - 저택 내부 규칙이 주인공에게 너무 좁아져 다음 무대를 요구할 때
  - 저택 파트 말미에 `최종 판정은 보류지만, 즉시 제거/보고는 하지 않겠다`는 신호가 주인공에게 전달될 때
  - 하급 과정에서 `무엇을 배워야 각인광장을 만들 수 있는가`에 대한 병목이 구체화될 때
  - 상급 과정 진입성취평가에서 상위권 우선권을 쥔 키리온이 입학 즉시 보존각인학파를 고르는 클리프행어가 필요할 때
  - 학술원 상급 과정 진입 후, 칼리온이 더 이상 관찰자가 아니라 판정자로 나서야 하는 가족 이벤트가 발생할 때
- load when:
  - academy pacing, 하급 과정/상급 과정 분리, 도입부 분량 배분, 외부 캐스트 투입 시점을 판단할 때
