# story_arcs

역할: 장기 연재에서 오래된 회차를 통째로 다시 읽지 않기 위해, 아크 단위로 압축된 서사 기억을 보관한다.

## Usage rule
- 기본 집필 시에는 `status: active` 아크만 읽는다.
- 각 아크는 "무슨 일이 있었나"보다 "지금 무엇이 열려 있나"를 중심으로 갱신한다.
- 5~10화 단위로 최근 `episode_deltas`를 흡수해 이 문서를 압축 갱신한다.

## ARC-001 정체 은폐와 칼리온의 관찰전
- status: active
- scope: `ep000` -> ongoing
- core question: 키리온은 아버지의 검증망 아래에서 얼마나 오래 정상적인 8세 차남으로 위장할 수 있는가.
- compressed memory:
  - `ep000`에서 칼리온은 질문 구조와 맥락 전환으로 키리온의 정합성을 테스트했다.
  - 키리온은 여덟 살짜리처럼 보이기 위해 답변 속도와 언어 수준을 강제로 낮추는 생존 규칙을 세웠다.
  - `ep001`에서 치유사 검사 자체는 통과했지만, 칼리온은 `흑색 표준식 입문서`를 보내 관찰의 시간을 길게 늘렸다.
- current state:
  - 위협은 즉석 심문에서 장기 관찰로 이동했다.
  - 앞으로는 읽기 습관, 이해 속도, 반응 지연까지 전부 리스크가 된다.
- next trigger:
  - 책을 읽는 장면에서 지식 노출/과소연기/과잉연기 중 어느 위험이 먼저 터지는가.
- load when:
  - 칼리온, 독서, 시험, 위장, 정체 노출 리스크가 중심일 때

## ARC-002 정보 병목 인식과 장기 야망의 현실화
- status: active
- scope: `ep000` -> ongoing
- core question: 주인공의 정보 인프라/커뮤니티 집착은 어떤 현실적 병목과 공동체 온도를 만나 행동 가능한 목표가 되는가.
- compressed memory:
  - `ep000`에서 주인공은 전생의 커뮤니티 유지보수 감각과 정보 흐름 설계 습관을 회수했지만, 장기 포부를 크게 선언하기보다 우선 생존으로 수렴했다.
  - `ep001`에서 집사실을 보며, 기록은 존재하지만 접근 인터페이스가 지나치게 좁다는 사실과 사람들끼리 문답을 부풀리며 노는 공동체 온도를 동시에 포착했다.
  - 주인공은 종이 게시판/쓰레드 구조까지 떠올렸지만, 아직 해결 설계로 진입하지 않았고 관찰의 순간에서 멈춰 있다.
- current state:
  - 장기선은 거창한 슬로건보다 "저택 내 정보 흐름 관찰 + 사람들이 답 주변에 모여드는 온도"라는 이중 발판을 얻었다.
  - 하지만 행동으로 옮기면 정체 노출 위험이 커서, 당장은 분석과 관찰에 머물러야 한다.
- next trigger:
  - 저택 내부 문서/전달/권한 체계가 추가로 드러날 때
  - 주인공이 작은 편의 개선 욕구나 임시 게시 체계를 떠올리지만 실행을 참아야 할 때
- load when:
  - 정보 인프라, 행정 병목, 커뮤니티 구축 장기선이 중심일 때

## ARC-003 학술원 입학 전 부트스트랩
- status: active
- scope: `ep000` -> academy entry
- core question: 학술원 입학 전까지 독자에게 무엇을 먼저 각인시켜야 이후 확장 서사가 무너지지 않는가.
- compressed memory:
  - 현재 서사는 넓은 세계를 바로 열지 않고, 렌바렌 저택이라는 좁은 공간에서 감시/돌봄/행정 구조를 먼저 세팅하고 있다.
  - `ep001`에서 학술원 입학까지 4년이 남았다는 시간축이 제시되어, 외부 무대가 당장 열리지 않는 이유가 구체화됐다.
  - NC 코어 캐스트 확장과 정식 시뮬레이션은 입학 이후로 미뤄진 상태다.
- current state:
  - 입학 전 파트의 임무는 주인공의 생존 규칙, 가문 분위기, 장기 공론장 욕망의 씨앗, 감시 구도를 고정하는 것이다.
  - 넓은 외부 캐스트를 선제 투입하지 않고, 내부 밀도를 높이는 편이 맞다.
  - 입학 목표 시점은 잠정적으로 `ep010` 전후다.
  - 다만 입학 전에는 `칼리온의 최종 확언`까지 가지 않는다. 저택 파트에서는 주인공이 `시험을 일단 넘겼다`는 가벼운 안도만 확보하면 충분하다.
  - `칼리온의 "그럼에도 내 아들이다" 확언`과 주인공의 더 깊은 정체성 수렴은 학술원 진입 후의 가족 이벤트로 미룬다.
- next trigger:
  - 입학 준비라는 외부 일정이 처음 서사 전면에 등장할 때
  - 저택 내부 규칙이 주인공에게 너무 좁아져 다음 무대를 요구할 때
  - 저택 파트 말미에 `최종 판정은 보류지만, 즉시 제거/보고는 하지 않겠다`는 신호가 주인공에게 전달될 때
  - 학술원 진입 후, 칼리온이 더 이상 관찰자가 아니라 판정자로 나서야 하는 가족 이벤트가 발생할 때
- load when:
  - academy pacing, 도입부 분량 배분, 외부 캐스트 투입 시점을 판단할 때
