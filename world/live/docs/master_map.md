## Path convention
- 이 문서의 `docs/...`, `world_bible/...`, `population/...`, `writing/...`, `quarantine/...` 표기는 모두 현재 bundle 루트인 `world/live/` 기준 상대경로다.
- 저장소 루트 기준 실제 경로가 필요하면 앞에 `world/live/`를 붙여 해석한다. 예: `docs/master_map.md` -> `world/live/docs/master_map.md`
