## 요약(선택)
- [PUBLIC] `마나 사인`은 개인의 고유 인증 수단이고, `하급 정령 계약`은 노드 인프라 접속권이며, `배지`는 둘을 실무적으로 묶어 쓰는 표준 단말이다.
- [PUBLIC] 마나 사인은 왕국의 공문, 계약, 등록, 책임 기록에 쓰이는 기초 인증 인프라라서 평민도 그 존재와 쓸모를 안다.
- [PUBLIC] 하급 정령 계약은 개인별 파트너 정령 계약이 아니라, 성목이나 영험지에 연결된 정령 체계를 이용하는 실무 계약이다.
- [PUBLIC] 배지는 신분표가 아니라 출입, 열람, 공지 수신, 기본 메시지 송수신, 기록 요청을 수행하는 네트워크 단말이다.
- [PUBLIC] 배지는 끝까지 `얇은 표준 단말`로 이해된다. 저장된 형식을 재해석하거나 새 감각을 만들어 내기보다, 권한 범위 안의 정보를 그대로 띄우는 쪽에 가깝다.
- [CONFIDENTIAL] 상위 로그 열람, 시설 제어, 광역 중계, 공문 발행, 감찰 추적 같은 기능은 별도 권한이나 상위 계약이 필요하다.

## 본문
### 5.1 세 층 구조

| 층 | 기능 | 공개 인식 |
| --- | --- | --- |
| 마나 사인 | 개인 인증 | 내 서명, 내 책임, 내 기록 |
| 하급 정령 계약 | 노드 인프라 접속 | 배지와 공공 마법 인프라를 쓰기 위한 접속권 |
| 배지 | 표준 단말 | 인증, 알림, 기록, 열람, 메시지, 출입 관리 장치 |

- [PUBLIC] 세상 사람들은 이 셋을 한 덩어리처럼 쓰지만, 실제 기능은 분리되어 있다.
- [PUBLIC] `마나 사인`이 없으면 본인 인증과 책임 귀속이 성립하지 않고, `하급 정령 계약`이 없으면 노드 인프라 접속이 불완전해지며, `배지`가 없으면 이를 실무적으로 다루기 어렵다.

### 5.2 마나 사인
- [PUBLIC] 마나 사인은 개인의 고유 정신적 서명이 담긴 표준 인증 수단이다.
- [PUBLIC] 왕국의 행정, 계약, 등록, 책임 기록은 기본적으로 정본과 마나 사인을 함께 요구한다.
- [PUBLIC] 마법사뿐 아니라 평민도 마나 사인의 존재와 쓸모를 안다. 관청, 상단, 길드, 학교, 서약원 모두 이를 활용한다.
- [PUBLIC] 마나 사인은 원래 더 배타적인 등록식 계열이었으나, 제도 정비와 대중화 끝에 사실상 누구나 쓰는 표준식으로 굳은 대표 사례로 여겨진다.

### 5.3 하급 정령 계약
- [PUBLIC] 하급 정령 계약은 배지, 기록, 전달, 인증 같은 인프라 사용을 위해 널리 익히는 표준식이다.
- [PUBLIC] 이 계약은 개인에게 하나씩 배정된 정령 친구를 만드는 계약이 아니라, 성목이나 영험한 노드와 연결된 정령 체계를 이용하는 실무 접속이다.
- [PUBLIC] 하급 정령은 대체로 기능 중심 존재로 여겨지며, 개성이나 이름이 거의 없는 편으로 받아들여진다.
- [PUBLIC] 하급 정령 계약 역시 원래는 등록식에 가까웠으나, 사용자가 폭증하면서 사실상 표준식으로 내려온 대표 사례로 인식된다.
- [CONFIDENTIAL] 실제로는 특정 노드가 운용하는 정령군 시스템에 접속권을 얻는 구조로 이해할 수 있다.

### 5.4 배지 네트워크
- [PUBLIC] 배지는 신분표가 아니라 `표준 마법 단말`이다.
- [PUBLIC] 배지는 정보 표시, 출입 인증, 열람 요청, 공지 수신, 기본 메시지 송수신, 실습 로그 기록을 한 번에 수행한다.
- [PUBLIC] 학내와 관청의 배지망은 대체로 `수심(Tree-Heart) -> 중계주 -> 배지` 구조로 이해된다.
- [PUBLIC] 수심은 노드 인프라의 핵심부이고, 중계주는 권역을 확장하는 릴레이 장치이며, 배지는 개인 단말이다.
- [PUBLIC] 배지는 권한 범위 안에서 MP 계측, 상태 표시, 경보 수신 같은 기능도 수행할 수 있다.
- [PUBLIC] 배지가 다루는 표준 출력은 텍스트, 표식, 간단한 도해, 허가된 경량 시청각 기록까지다. 복합 감각 재생은 별도 고정식 설비의 영역으로 받아들여진다.
- [CONFIDENTIAL] 기록 설계와 구조 편집을 직접 수행하는 `그리모어류 관리자 콘솔`은 배지와 다른 계층의 도구다.

### 5.5 하급 정령 계약으로 가능한 기본 기능
- [PUBLIC] 배지 활성화
- [PUBLIC] 신원 인증과 접속 인증
- [PUBLIC] 학내 공지와 긴급 경보 수신
- [PUBLIC] 기본 메시지 송수신
- [PUBLIC] 권한 확인
- [PUBLIC] 정본, 로그, 시설에 대한 기본 열람 요청 전달
- [PUBLIC] 각인광장, 학내 네트워크, 중계주 같은 노드 인프라 접속
- [PUBLIC] 허가된 범위 안의 경량 기록 열람과 영상 각인 재생

### 5.6 별도 권한이나 상위 계약이 필요한 기능
- [CONFIDENTIAL] 상위 로그 열람
- [CONFIDENTIAL] 시설 제어
- [CONFIDENTIAL] 장거리 또는 광역 중계
- [CONFIDENTIAL] 공문 발행
- [CONFIDENTIAL] 타인 기록 열람
- [CONFIDENTIAL] 감찰이나 징계용 추적
- [CONFIDENTIAL] 대규모 방송, 실시간 공공 중계
- [CONFIDENTIAL] 고감각 현상 재생과 고정식 열람실 운용
- [CONFIDENTIAL] 영상 각인 업로드 권한과 촉매 승인
- [CONFIDENTIAL] 관리자 콘솔을 통한 기록 구조 편집과 공개 범위 결정

### 5.7 범위 밖
- [PUBLIC] 하급 정령 계약은 전투용 정령 소환을 직접 보장하지 않는다.
- [PUBLIC] 개별 정령과의 인격적 교류, 자율 판단형 보좌 정령 운용, 중급 이상 정령과의 본격 계약은 별개 문제다.
- [PUBLIC] 따라서 배지 계약이 곧 정령술사의 본격 계약을 뜻하지는 않는다.

### 5.8 사회적 위상
- [PUBLIC] 일반인도 마나 사인과 배지의 존재를 알고, 이를 공인 인증과 행정 인프라의 일부로 여긴다.
- [PUBLIC] 하급 정령 계약은 학생, 관료, 실무자에게 특히 익숙한 개념이며, 성목이나 노드 인프라가 발달한 지역일수록 체감도가 높다.
- [PUBLIC] 각인광장, 학내 게시판, 기록실, 제한 구역 출입, 실습 로그 관리 같은 장치는 모두 이 구조 위에서 돌아간다.
- [PUBLIC] 따라서 세계관 상식 차원에서도 `배지 업그레이드 하나로 만능 감각 체험이 열린다`기보다, `필요한 기록은 배지로 보고 더 위험한 기록은 별도 시설에서 본다`는 감각이 더 자연스럽다.
