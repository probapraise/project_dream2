## 요약(선택)
- [CONFIDENTIAL] `무흔술`은 기사계 품새 안에 숨겨 운용하는 `수경원`의 비마나 실전 전투술 정본이다.
- [CONFIDENTIAL] 겉보기 동작은 제도권 기사 무예와 닮아도, 실제 판정 기준은 `선수 확보 -> 축 붕괴 -> 짧은 결착 -> 즉시 이탈`이다.
- [CONFIDENTIAL] 칼리온은 정본을 데리온에게 교리와 금기까지 포함해 전수하지만, 아직 열 살인 데리온이 키리온에게 넘길 수 있는 것은 `전날 교정을 다음날 형제 대련으로 되풀이하는 압축본`뿐이다.
- [CONFIDENTIAL] 그래서 현재 키리온이 익히는 것은 조직용 암살 교범이 아니라, 데리온형 압축본을 자기 체구와 분석 습관에 맞게 최적화한 비정통 변형이다.
- [CONFIDENTIAL] 치유사 상주와 과훈련 덕분에 키리온의 훈련 밀도는 비정상적으로 높지만, 성장판과 체급의 시간 자체는 건너뛸 수 없다. 그래서 10~12세 구간의 상승은 절대근력보다 `반응 / 체간 안정 / 검선 효율 / 순간 가속`에 먼저 쌓인다.

## 본문
### I.1 정의와 목적
- `무흔술`은 제도권 기사 무예의 대체물이 아니라, 그 외형을 빌려 운용하는 은폐형 실전 체계다.
- 목적은 `마나를 쓰지 않고도 살아남고, 필요하면 짧게 제압하거나 죽이고, 들키기 전에 빠지는 것`이다.
- 그러므로 `무흔술`의 우선순위는 기사도나 시합 승리가 아니라 다음 순서로 잡는다:

| 우선순위 | 정본 판정 기준 |
| --- | --- |
| 1 | 살아남을 것 |
| 2 | 마나 흔적과 증언을 최소화할 것 |
| 3 | 몸의 축을 먼저 무너뜨릴 것 |
| 4 | 짧은 박자 안에 결착하거나 끊고 빠질 것 |
| 5 | 끝난 뒤 머무르지 않을 것 |

- 이 체계는 결투용, 시범용, 명예용 무예가 아니며, `침투 / 입막음 / 긴급 생존 / 호위 실패 시 탈출` 같은 심부 실무를 전제로 한다.

### I.2 기사 무예와의 차이

| 항목 | 제도권 기사 교범/귀족 자제 무예 | 정본 `무흔술` |
| --- | --- | --- |
| 겉모습 | 품새, 초식, 반복 동작 | 품새와 닮은 동작을 빌려 씀 |
| 목적 | `마나 + 식`이 몸을 따르게 하는 의례화 | 마나 없이도 상대를 짧게 무너뜨림 |
| 판정 기준 | 자세, 리듬, 초식 재현 | 첫 박자, 각도, 축 붕괴, 이탈 |
| 이상적 합 | 몇 합이고 이어 가도 됨 | 세 박자 안에 우세, 아니면 즉시 끊음 |
| 마무리 | 제압, 승부, 기량 증명 | 생존, 은폐, 결착, 이탈 |

- 그래서 외부 관찰자에게는 비슷해 보여도, 가까이서 듣는 교정 언어와 동작 목적은 완전히 다르다.

### I.3 정본 5원칙
1. `겉은 품새, 속은 결착`
   공개 교범처럼 보여도 실제 목적은 제압과 살상, 그리고 비노출이다.
2. `정면으로 받지 않는다`
   막아 내는 힘싸움보다 비껴서며 선을 바꾸는 움직임을 우선한다.
3. `검보다 축을 먼저 건드린다`
   손보다 손목, 손목보다 어깨, 어깨보다 중심축을 먼저 무너뜨린다.
4. `세 박자 안에 끝낸다`
   첫 박자에 우세를 못 잡으면 길게 버티지 않고 리듬을 끊고 빠진다.
5. `끝난 뒤 머물지 않는다`
   적을 쓰러뜨린 뒤 멈춰 서는 것은 실수다. 정본은 결착과 이탈을 한 세트로 본다.

### I.4 정본 훈련 층
- `보법`: 곧게 물러서지 않고 반 박자 비껴서는 발.
- `선탈취`: 상대 검선 정면에 남지 않고 자기 선만 남기는 각도 감각.
- `구조붕괴`: 손목, 팔꿈치, 어깨, 무릎, 중심축처럼 작은 힘으로 흐트러뜨릴 지점 우선순위.
- `짧은 결착`: 긴 합을 피하고 1~3박 안에 결과를 만드는 습관.
- `이탈`: 끝낸 뒤 멈추지 않고 시야, 위치, 태세를 숨기는 후속 움직임.

- 정본 교본은 여기서 더 나아가 `침투 동선`, `입막음 방식`, `사후 정리`, `조직 보안`, `식흔`과의 병행 기준까지 포함하지만, 그것은 심부 실무자 층위다.

### I.5 전수 병목
- `칼리온 -> 데리온`
  - 정본 원리, 금기, 실패했을 때 왜 죽는지, 왜 이 전승이 `수경원` 비밀인지까지 포함한다.
  - 즉 `무흔술`을 단순 검술이 아니라 비공개 실무 체계로 배운다.
- `데리온 -> 키리온`
  - 현재 시점에서는 정식 교본 강의가 아니다.
  - 데리온은 전날 칼리온에게 교정받은 발, 각도, 첫 박자, 붕괴 지점을 다음날 키리온과의 대련으로 다시 때려 넣는다.
  - 따라서 키리온에게 남는 것은 `왜`보다 `어떻게 안 죽는가`에 가까운 몸의 답이다.

- 열 살 데리온이 현재 가르칠 수 없는 것:
  - 조직용 암살 프로토콜
  - 침투/시체 처리/사후 정리
  - 정치적 맥락과 침묵계약의 전모
  - `식흔`과 `무흔술`의 조합 운용 전체

### I.6 데리온형 `무흔술`
- 정본이 후계자 공포와 보호 강박에 눌려 압축된 변형이다.
- 특징:
  - 설명보다 교정이 많다.
  - 첫 박자 선점과 버티기에 과하게 집착한다.
  - 예쁜 동작보다 "일단 안 죽는 버릇"을 먼저 박아 넣으려 든다.
  - 작은 실수에도 압력을 크게 걸어 실제 상황처럼 몰아붙인다.
- 강점:
  - 거리 압박
  - 첫 박자 반응
  - 맞아도 자세를 안 무너뜨리는 버티기
  - 기본 구조 붕괴 감각
- 약점:
  - 감정이 섞여 과교정되기 쉽다.
  - 은밀 공작 전체 체계로는 아직 미완성이다.
  - 사사라기보다 폭력적 복습에 가까워 전수 효율이 불안정하다.

### I.7 키리온형 `무흔술`
- 키리온은 이를 비밀 전승이 아니라 `그냥 쓸 만한 검술`로 받아들인다.
- 이 오해 때문에 오히려 동작의 권위보다 `진짜 먹히는가`를 먼저 본다.
- 특징:
  - 작은 체구에 맞춰 각도와 타이밍을 더 아낀다.
  - 형이 시키는 동작을 그대로 숭배하지 않고, 패턴을 읽어 자기 몸에 맞게 다시 줄인다.
  - 후일에는 `열상각인` 같은 일부 보조 마법을 붙여 `무흔술` 한 번 더 먹이기 위한 보조 장치로 쓴다.
- 강점:
  - 읽기와 대응 최적화
  - 작은 움직임으로 결과를 내는 감각
  - 비전투 마법과의 접합 여지
- 약점:
  - 암살자적 교리와 조직 실무를 모른다.
  - 자기 전투력의 희소성과 위험성을 과소평가한다.
  - 살의를 전제로 한 완성형 판단 체계는 아직 없다.

### I.8 형제 대련형 쌍방상승
- 이 구조는 정식 사제 관계보다 `형제`라서 성립한다.
- 데리온은 동생을 봐주지 못한다. 보호 강박, 질투, 오기가 대련 압력을 높인다.
- 키리온은 형의 권위를 신성시하지 않는다. 그래서 설명을 기다리기보다 반복 패턴에서 답을 뽑아낸다.
- 결과적으로:
  - 데리온은 `칼리온에게서 배운 정본`을 작은 체구와 다른 사고방식에도 먹히는 형태로 다시 실전화한다.
  - 키리온은 `형에게서 맞아 가며 배운 압축본`을 자기 몸과 감각에 맞는 비정통 변형으로 재컴파일한다.

### I.9 현재 시점 적용 범위 (`pre-academy / 8세`)
- 지금 키리온이 실제로 손에 넣은 것은 아래 층위가 중심이다:
  - 곧게 물러서지 않는 발
  - 첫 박자에 비껴서는 각도
  - 손목/어깨/중심축 같은 붕괴 지점 감각
  - 길게 합을 끌지 않고 짧게 끝내려는 습관
  - 맞고도 다시 일어나는 버티기
- 아직 없는 것은 아래 층위다:
  - 조직용 암살 실무
  - 침투와 사후 정리
  - 독, 교살, 시체 은닉 같은 특수 프로토콜
  - `수경원` 내부 명령 체계와 비밀 전승의 정치적 의미

- 따라서 `ep003` 전후의 외부 표면은 그냥 `형제가 목검으로 거칠게 대련하는 장면`처럼 보여야 한다.
- 이때 차이를 만드는 것은 교정 기준이다. 정본이 새는 표면 표식은 `자세가 예쁜가`보다 `곧게 물러섰는가`, `검을 봤는가`, `쳤는데 멈췄는가` 같은 판단 언어 쪽에 있다.

### I.10 치유 세계의 과훈련과 피지컬 상한
- 렌바렌 같은 상위 귀족가는 영양, 온열 관리, 수면 환경, 회복용 약재, 경상 치료, 관절 관리에 모두 유리하다.
- 여기에 치유사가 상주하면 타박, 염좌, 근육 미세 파열, 손목과 발목의 반복 손상을 빠르게 회복시킬 수 있어 `훈련 빈도`와 `반복 횟수`가 비정상적으로 높아진다.
- 하지만 치유사는 아래 항목을 건너뛰게 만들지 못한다:
  - 성장판이 닫히지 않은 골격의 한계
  - 짧은 팔다리와 작은 체중이 만드는 지렛대 한계
  - 미성숙한 흉곽과 골반, 등판 두께
  - 체중 자체가 부족해 생기는 순수 파괴력 부족
- 따라서 어린 사용자의 `무흔술`은 성인보다 먼저 아래 순서로 강해진다:

| 먼저 빨라지는 것 | 늦게 따라오는 것 |
| --- | --- |
| 반응 속도 | 절대근력 |
| 발/각도 전환 | 체중 실린 압박 |
| 체간 안정 | 갑주 상대 파괴력 |
| 검선 정밀도 | 장기전 지속력 |
| 고통 내성과 회복 루틴 | 큰 체구 상대로 버티는 힘 |

- 이 차이 때문에 10~12세 키리온의 전투력은 `기술 곡선 > 피지컬 곡선` 구조를 띠어야 한다.
- 반대로 11세 후반~12세 무렵 몸이 조금씩 따라붙기 시작하면, 이미 누적된 기술 밀도가 비로소 실제 살상력으로 번역된다. `ep010`의 반전은 바로 이 지점에서 터지는 편이 논리적이다.

### I.11 데리온 academy 부재기 (`키리온 10~12세`) 입력 구조
- `독자 수련`
  - 보법 반복
  - 그림자 검선
  - 벽/기둥/줄을 이용한 거리 재기
  - 손목/전완/악력 보강
  - 체간 고정과 회전 연결
  - 짧은 폭발 후 호흡 회복 루틴
- `리리아를 가르치는 과정`
  - 리리아가 자기도 체술을 배우고 싶다며 달라붙는다
  - 키리온은 위험한 살상 동작을 빼고 호신술처럼 축약해 보여 준다
  - 붙잡힌 손 빼기, 거리 벌리기, 동행자 이탈 같은 동작을 `설명 가능한 형태`로 쪼개게 된다
  - 작은 상대와 느린 파트너 기준으로 반복하다 보니, 보호 대상을 끌고 움직이는 발과 속도 조절 감각이 붙는다
  - 결과적으로 키리온형 `무흔술`은 살상술만이 아니라 `호신 / 호위 / 이탈` 쪽 운용 문법이 함께 정리된다
- `방학마다 돌아오는 데리온`
  - 몇 주간 강도 높은 대련으로 잘못된 습관을 부순다
  - 키리온이 혼자 만든 오답을 제거한다
  - 유효했던 최적화만 남긴다

- 이 2년 동안 키리온은 새 교범을 받지 않는다.
- 대신 `과거 입력된 압축본 + 자기 분석 + 리리아를 가르치며 생긴 언어화/호위 감각 + 데리온의 계절성 재교정`이 결합해, 정본과는 다른 `키리온형 무흔술`이 서서히 닫힌다.

### I.12 키리온 10~12세 피지컬 성장 곡선

| 시점 | 신체 변화 | 전투적 의미 | 아직 없는 것 |
| --- | --- | --- | --- |
| 10.0 | 여전히 어린 체구. 절대 체중과 팔 길이가 부족하다 | 발, 각도, 반응은 또래 이상으로 빠르다 | 방어를 힘으로 부수는 출력 |
| 10.5 | 발목/무릎/골반 연결이 안정되고, 체간이 덜 흔들린다 | 첫 박자 스타트와 정지-재가속이 눈에 띄게 좋아진다 | 큰 상대를 밀어내는 힘 |
| 11.0 | 전완, 악력, 등판 하부, 복압 유지가 눈에 띄게 좋아진다 | 검끝이 덜 떨리고, 짧은 타격도 더 정확히 박힌다 | 갑주나 두꺼운 외피를 억지로 가르는 힘 |
| 11.5 | 체중 전달과 회전 연결이 붙기 시작한다 | 약점에 정확히 꽂을 때 사람과 소형 몬스터를 크게 무너뜨릴 수 있다 | 정면 힘싸움과 장기전 우세 |
| 12.0 | 몸이 그간 축적된 기술을 실전 출력으로 번역할 최소선을 넘는다 | `짧은 거리 + 정확한 약점 + 완전한 타이밍` 조건이면 일격 결착이 가능해진다 | 성인 체급, 다수전 지구력, 본격 마법전 대응 |

- 핵심은 12세 키리온이 `갑자기 강해진 것`이 아니라, 이미 누적된 기술이 몸의 성장으로 뒤늦게 살상력까지 번역되기 시작했다는 점이다.

### I.13 12세 키리온 전투력의 적정 상한
- 또래 일반 귀족과의 순수 근접전: 숨겨진 최상위권
- 형식 훈련 위주 또래 기사 지망생: 명확 우세
- 선행학습 서명귀족 또래: `좁은 거리 / 첫 시전 전 / 얕봄` 조건이면 충분히 이길 수 있음
- 방심한 성인 비전문가나 하급 경비: 매우 위험함
- 훈련된 성인 기사, 실무 암살자, 갑주 사용자: 아직 안정적으로 안 됨
- 다수전, 개활지, 선시전 허용 마법전: 한계가 분명함

- 따라서 12세 키리온의 위력은 `완성된 성인 암살자`가 아니라, `또래 인간형 근접전에서는 괴물이지만 세계 전체 기준으론 편향된 비대칭 자산`으로 운용하는 편이 맞다.
