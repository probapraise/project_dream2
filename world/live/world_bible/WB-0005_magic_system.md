## 요약(선택)
- [PUBLIC] 프라미시오의 정형화 마법은 실무 문법상 `색 × 형 × 잔류`로 읽는다.
- [PUBLIC] 형은 `사출 / 전개 / 피복 / 기동 / 표식 / 탐지` 6형만 쓴다.
- [PUBLIC] `복원`, `판독`은 형이 아니라 효과 태그이며, `각인`은 더 이상 형 이름이 아니다.
- [PUBLIC] 표준식은 교양과 실습을 위해 정형화된 공용식이고, 등록식은 권한·로그·실패 모드까지 포함한 제도화 마법이다.
- [PUBLIC] 문장비전(Crest Arcana)은 혈통키와 창시자 서명에 묶인 배타적 등록식 계열이지만, 읽는 문법 자체는 `색 × 형 × 잔류`를 따른다.

## 본문
### 3.1 마나와 마나핵(MP)
- [PUBLIC] 마나는 `내계(內界: 신체/소우주)`와 `외계(外界: 바깥 현상)`를 매개하는 에너지이자 규칙 자원이다.
- [PUBLIC] 모든 시전자에게는 마나핵(코어)과 MP가 있으며, 배지 계측값(MAX MP/CURRENT MP/회복률 등)은 학칙과 권한 등급에 따라 공개 범위가 제한된다.
- [PUBLIC] 시전 효율은 단순 총량보다 `주색 적합`, `서명 정합`, `문장 숙련`, `잔류 유지력`에 크게 좌우된다.

### 3.1-A 벨쿠란 왕국의 아동 마법 금지 규정
- [PUBLIC] 벨쿠란 왕국은 `계승조회식`을 마치지 않은 아동에게 체계적 마법 수련을 시키는 것을 금지한다.
- [PUBLIC] 마법의 분류, 절차, 이론, 정형문을 가르치는 것은 허용되지만, 아동 스스로 마법을 사용할 수 있도록 마나 회로와 마나핵 활성화를 보조하는 행위는 엄금된다.
- [PUBLIC] 적발 시 부모는 반역죄에 준하는 형을 구형받고, 해당 영지 전체가 고강도 특별조사에 들어간다.
- [PUBLIC] 첫 사용 이후에는 순환 리듬, 미세 잔향, 회로 개방 흔적이 감정에 포착되므로, 단순 부인만으로는 은닉이 어렵다.
- [CONFIDENTIAL] 완전한 은닉이 불가능한 것은 아니나 고비용·고난도 기만이 필요하고, 오히려 은닉술 흔적이 따로 잡히는 경우가 많다.
- [CONFIDENTIAL] 왕실·공작가·일부 2시그니처 이상 가문은 사실상 조기 수련 특권을 누리며, 이 비대칭이 하급 과정 입학 직후부터 성취 격차로 드러난다.

### 3.2 실무 문법: `색 × 형 × 잔류`
- [PUBLIC] `색 × 형 × 잔류`는 초월력 전체를 덮는 우주론이 아니라, 표준식·등록식·문장법·학파식을 가르치고 기록하기 위한 실무 문법이다.
- [PUBLIC] 주문은 기본적으로 `주색 1개 + 형 1개 + 잔류 1개`로 읽고, 필요할 때 효과 태그 1~3개를 덧붙인다.
- [PUBLIC] 예시: `적 / 사출 / 순간 / 가열`, `청 / 전개 / 유지 / 구획`, `자 / 표식 / 유지 / 봉문`, `백 / 탐지 / 흔적 / 검증`.

#### 3.2.1 색(7색)

| 주색 | 핵심 경향 | 대표 용례 |
| --- | --- | --- |
| 적 | 열, 압력, 축열, 폭발 | 포격, 절단, 과열, 붕괴 유도 |
| 청 | 구조, 구속, 하중, 구획 | 방벽, 고정, 경계선, 통과 규칙 |
| 녹 | 생명, 약리, 생태, 여과 | 응급처치, 독성 운용, 포자장, 진단 |
| 황 | 운동, 탄도, 편향, 박자 | 돌진, 넉백, 궤도 제어, 전열 유도 |
| 자 | 교신, 기만, 우선도, 왜곡 | 전심, 환영, 주목도 조정, 기억 흔적 읽기 |
| 백 | 정화, 가호, 승인, 제재 | 오염 제거, 보호막, 허가, 서약 집행 |
| 흑 | 침식, 은폐, 결손, 단절 | 부식, 맹점, 잠행, 소환선 절단 |

#### 3.2.2 형(6형)

| 형 | 기본 동사 | 대표 표면 |
| --- | --- | --- |
| 사출 | 쏘아 보내 맞힌다 | 탄, 광선, 파편, 침 |
| 전개 | 면과 구역을 펼친다 | 장판, 벽, 안개, 경계선 |
| 피복 | 대상에 둘러씌운다 | 막, 외피, 갑주, 필터 |
| 기동 | 밀고 당기고 옮긴다 | 돌진, 도약, 견인, 편이 |
| 표식 | 찍고 새기고 조건을 남긴다 | 낙인, 인장, 룬선, 봉문선, 지시점 |
| 탐지 | 숨은 정보와 흔적을 드러내 읽는다 | 주사선, 반응면, 추적선, 유령 지도 |

#### 3.2.3 잔류(3축)

| 잔류 | 의미 | 운용 메모 |
| --- | --- | --- |
| 순간 | 바로 발동하고 사라진다 | 폭딜, 절단, 짧은 교란에 적합 |
| 유지 | 일정 시간 버틴다 | `부착`, `설치`, `체류` 등 운용 태그를 자주 동반 |
| 흔적 | 결과가 잔류층에 남는다 | 역추적, 검증, 공공 로그와 결합하기 쉽다 |

#### 3.2.4 효과 태그와 legacy naming
- [PUBLIC] `복원`, `판독`, `검증`, `회복`, `정화`, `유폭` 등은 형이 아니라 결과를 설명하는 효과 태그다.
- [PUBLIC] `각인`은 더 이상 형 이름이 아니다. 다만 `마나사인`, `연장각인`, `발췌각인`, `영상 각인`, `각인광장`, `보존각인학파` 같은 기술명·기관명·전통어에는 남길 수 있다.
- [PUBLIC] 위와 같은 legacy 용어도 실제 분류 설명에서는 반드시 `표식` 또는 `탐지`로 재해석한다.

### 3.3 표준식 vs 등록식 vs 문장비전
- [PUBLIC] 표준식은 교양과 실습을 위해 정형화된 공용식이다. 하급 과정에서는 색 이전에 `형을 읽는 법`을 가르치고, 이후 주색 입문서와 공용 표식 기술을 얹는다.
- [PUBLIC] 등록식은 특정 학파·기관·가문이 구축·등록한 세계 기능이다. 절차, 권한, 실패 모드, 감독 로그까지 포함하므로 사회 인프라로 취급된다.
- [PUBLIC] 강한 등록식일수록 `누가`, `어떤 권한으로`, `어떤 조건에서` 발동했는지가 더 선명하게 남는다.
- [PUBLIC] 문장비전(Crest Arcana)은 창시자 서명과 혈통키에 묶인 배타적 등록식 계열이다. 희소성과 독점성이 곧 권력이며, 표현 문법은 여전히 `색 × 형 × 잔류`를 따른다.

### 3.4 실무 운용 규칙(정본 3.1 기준)
- [PUBLIC] 강한 출력은 더 선명한 조건과 더 무거운 비용을 요구한다.
- [PUBLIC] 강한 표식형은 조건·매개·권한선이 읽히면 약해진다. 감춘 표식은 오래 숨지만 약하고, 선명한 표식은 강하지만 해제 단서도 크다.
- [PUBLIC] 탐지형은 차폐, 잡음, 세정, 위장, 역탐지에 흔들린다. 완벽한 탐지는 금지하며, 항상 거짓 양성과 거짓 음성의 위험을 남긴다.
- [PUBLIC] 유지형은 반드시 지워질 수 있어야 하고, 흔적형은 반드시 비용·시간·감독 위험을 동반한다.
- [PUBLIC] 흑색 상급식과 상급 문장법, 일부 왕실 계전식은 발동 사실 자체가 감독망·공공 로그에 남는다.

### 3.5 하급 과정 공통 문법과 교양화된 실무
- [PUBLIC] 하급 1학년의 목표는 `지금 본 주문이 무슨 색이고, 어떤 형이며, 어떻게 남는지`를 말할 수 있게 만드는 데 있다.
- [PUBLIC] 하급 2학년에서는 주색 입문서와 함께 `사출 / 피복 / 표식` 기초를 먼저 익히고, 2학기에 `전개 / 기동 / 탐지`를 완주한다.
- [PUBLIC] 하급 2학년 말에는 `마나사인`, `연장각인`, `정본해시`, `단순봉문` 같은 공용 표식 기술이 사실상 교양 수준으로 들어간다.
- [PUBLIC] 이 교양화 때문에 보존각인 계열의 옛 실무는 탈희소화되었고, 보존각인학파는 `부전공 값은 높지만 주전공 값은 죽은 학파`라는 인식을 뒤집어쓰게 되었다.

### 3.6 상급 과정 마법사 21학파(7탑 × 3학파)
- [PUBLIC] 아르케이온 상급 과정의 마법사 전공 SSOT는 `21학파`다.
- [PUBLIC] 각 학파는 보통 `입문 → 초급 → 중급 → 상급 → 오의`의 5단계 전승을 가지며, 전승 수는 운영상 12식을 기본선으로 본다.

| 탑 | 학파 | 핵심 역할 | 주 수요처 |
| --- | --- | --- | --- |
| 적 | 열압포격학파 | 원거리 포격 / 공성 | 군, 포대, 제련소 |
| 적 | 용융형상학파 | 장비 파괴 / 즉석 무장 | 단조, 공방, 병기 |
| 적 | 연무통제학파 | 시야 차단 / 재난 대응 | 소방, 환기, 재난 대응 |
| 청 | 성벽수호학파 | 거점 유지 / 호위 | 도시결계, 건축, 호위 |
| 청 | 결박제어학파 | 구속 / 시전 차단 | 치안, 구금, 수송 |
| 청 | 좌표감리학파 | 지형 교정 / 탈출 봉쇄 | 관문, 교량, 측량 |
| 녹 | 응급약리학파 | 전장 의무 / 독 운용 | 치료원, 약리, 구호 |
| 녹 | 생태계약학파 | 정령 지원 / 분산 피해 | 산림, 성역, 계약 관리 |
| 녹 | 균사환경학파 | 구역 지배 / 생태전 | 농업, 생태 복원 |
| 황 | 탄도관성학파 | 저격 / 탄막 / 넉백 | 수송, 포병, 투사체 조율 |
| 황 | 자계유도학파 | 금속 제어 / 전기 연쇄 | 동력실, 공방, 장비 |

| 자 | 교신전심학파 | 교신 / 정신 교란 | 교신망, 전령, 지휘 |
| 자 | 허상연출학파 | 환영 / 미끼 / 기만 | 외교, 선무, 보안 기만 |
| 자 | 보존각인학파 | 보존 / 색인 / 재현 / 증거화 | 서고, 정본실, 박물고, 감식청 |
| 백 | 생명고정학파 | 생존 고정 / 구조 | 치료원, 구조대, 응급소 |
| 백 | 정화광휘학파 | 오염 제거 / 환영 파훼 | 성소, 정화실, 재난 수습 |
| 백 | 서약집행학파 | 해제 / 반사 / 제재 | 성소 경비, 허가 감찰, 계약 집행 |
| 흑 | 침식주술학파 | 장기전 / 부식 / 저주 | 위험물 해체, 저주 유물 처리 |
| 흑 | 음영잠행학파 | 은폐 / 맹점 / 침투 | 제한구역 경비, 비밀 작전 |
| 흑 | 단절추방학파 | 차단 / 추방 / 연결 절단 | 금고 봉쇄, 소환체 처리, 금역 관리 |

### 3.7 직능 3축: 마법사 / 기사 / 사제
- [PUBLIC] 마법사(Mage): `색 × 형 × 잔류`를 실전 문법으로 가장 직접 운용하는 축이다. 학파 선택과 등록식 설계·등록이 핵심이다.
- [PUBLIC] 기사(Knight): 내계 제어와 신체 강화, 장비 운용, 전술 기동에 특화된 축이다. 실전에서는 마법 문법을 읽되, 자체 훈련 언어는 별도로 유지한다.
- [PUBLIC] 사제(Priest): 서약과 의례, 정화와 집행에 특화된 축이다. 등록식과 권한 질서, 허가 구조를 다루는 비중이 크다.
- [PUBLIC] 세 직능은 동일한 마나핵/MP 체계를 공유하지만, 훈련 언어와 면허 체계가 다르다.

### 3.8 서명귀족과 문장비전(Crest Arcana)
- [PUBLIC] 문장비전은 `시스템이 기억할 가치가 있다`고 판정한 창시자 서명을 기반으로, 혈통키(가문 문장 서명 시드)로만 발동 가능한 등록식 계열이다.
- [PUBLIC] 대부분의 가문은 안정 전승 시그니처 1개(일부 2, 왕가 3)를 유지하며, 희소성과 독점성이 곧 권력·갈등이 된다.
- [PUBLIC] 왕실과 상위 귀족은 문장비전의 배타성뿐 아니라, 그 발동 기록과 감독망 자체를 권력 자산으로 쓴다.
- [CONFIDENTIAL] 흑색 상급식과 문장비전 상급 운용은 비밀리에 쓰더라도 감독 비용에서 완전히 자유롭지 않다.
