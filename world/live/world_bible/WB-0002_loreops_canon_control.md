## 요약(선택)
- [PUBLIC] 초반 인물/독자가 ‘세상 상식’으로 받아들이는 정보.
- [CONFIDENTIAL] 특정 기관·상층부만 아는 정보. 작품 전개에 따라 단계적으로 노출.
- [META] 작가 전용 ‘세계의 진실’. 작중 직접 노출은 제한(최종부/후일담 등).
- 불변식(Invariants): 바꾸면 연쇄 수정이 폭발하는 핵심 법칙(예: 마나사인 유통기한/갱신 규칙, 권한 경제(열람권·면허·접속권·영상 업로더 할당량·현상 열람 자격), 커뮤니티 레이어와 UI 공간의 분리 원칙).

## 본문
### 0.1 정보 공개 레벨(스포일러/노출 규칙)
- [PUBLIC] 초반 인물/독자가 ‘세상 상식’으로 받아들이는 정보.
- [CONFIDENTIAL] 특정 기관·상층부만 아는 정보. 작품 전개에 따라 단계적으로 노출.
- [META] 작가 전용 ‘세계의 진실’. 작중 직접 노출은 제한(최종부/후일담 등).
### 0.2 캐논 레이어(불변/안정/주장)
- 불변식(Invariants): 바꾸면 연쇄 수정이 폭발하는 핵심 법칙(예: 마나사인 유통기한/갱신 규칙, 권한 경제(열람권·면허·접속권·영상 업로더 할당량·현상 열람 자격), 커뮤니티 레이어와 UI 공간의 분리 원칙).
- 안정 캐논(Stable): 대체로 유지하지만 수정 가능(영향 분석 필수).
- 주장/루머(Claim): 논쟁적·가변적·스포일러성 정보. 모순이 ‘오류’가 아니라 ‘세계 내부의 충돌’이 되도록 관리.
### 0.3 증거 등급(권장 메타)
- S: 원본 정본(관청 접수) 또는 다중 검증된 핵심 기록.
- A: 강한 근거(감정/다수 로그/보존각인 포함).
- B: 그럴듯하나 빈틈 존재(맥락/증거가 부분적).
- C: 소문/단편 증언(커뮤니티 렌즈에 강하게 의존).
- D: 조작/선동 가능성이 높은 정보(혹은 의도적 미끼).
- 추가 메타: valid_from/valid_to(유통기한), source(출처), visibility([PUBLIC]/[CONFIDENTIAL]/[META]).
### 0.4 ID/명칭 운영(요약)
- ID는 불변(참조 무결성). 표시명(Display Name)은 변경 가능.
- [TEMP]는 임시 표시명. ID와 언어권(네이밍 키트)만 먼저 고정하고, 이름은 나중에 패치로 교체한다.
- 공식명(별칭) 병기를 허용하되, 별칭 매핑 표에 등록해 충돌을 방지한다.

### 0.5 장면 정합성 동적 체크 템플릿
- 장면에서 등장하는 핵심 정보마다 `visibility([PUBLIC]/[CONFIDENTIAL]/[META])` 표기가 있는지 확인한다.
- 장면의 사실 판단 근거에 `evidence_grade(S/A/B/C/D)`와 `valid_to`가 반영됐는지 확인한다.
- 권한(정본/열람권/배지/기관 권한)이 갈등 원인 또는 해결 비용으로 일관되게 작동하는지 점검한다.
- 사용된 마법(표준식/등록식/문장비전)의 권한 범위, 금기, 카운터 수단이 앞선 설정과 모순되지 않는지 확인한다.
- 필요 시 컨텍스트를 확장 점검한다: `docs/community_map.md` 활성 구조, 기관 규정 문서, 이벤트 로그/제재 기록.

### 0.6 운영 경계
- 본 문서는 장면 점검 규칙을 정의하며, 특정 보드 수/존 수 같은 고정 구조를 강제하지 않는다.
- 고정 구조/샘플 기반 레거시 규칙은 `quarantine/`에 보존하고 활성 캐논에는 직접 반영하지 않는다.
