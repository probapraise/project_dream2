### A.1 핵심 용어 매핑

| 구용어(v2.3) | 현행(v3.0) | 비고(원고 사용 규칙) |
| --- | --- | --- |
| 각인사인 / 마나서명 | 마나사인 | 원고에서는 '마나사인'만 사용 |
| 레거시 미디어 | 공보회 | 학내 하부조직은 '학생회 공보국'으로 단순화 |
| 광장석 | 성목 수심(樹心) / 코어 | 저장 코어는 '나무의 심장' 개념으로 통일 |
| 중계석(용어 잔재) | 중계주(中繼柱) | 돌이 아니라 기둥/기지국 이미지로 통일 |
| 프라미시오 제국 | [TEMP] 벨쿠란 왕국 | 주인공 소속은 왕국. 게이트 대륙엔 제국 존재 |
| 고유 마나 | 마력 / 마력 | 외부에 깔린 마나는 '대기마나', 개인 축적분은 '마력'. 대사에선 '마력' 허용. |
| 체내 코어 / 마나코어(개인) | 마나핵 | 인프라 코어(성목 수심)와 혼동 방지. 원고 표기는 '마나핵'으로 고정. |
| 가문 고유 학파 | 문장비전(혈통 잠금 학파) | 혈통키가 없는 자는 소속/사용 불가. 가문 비기는 3~7개 규모 권장. |
| 각인형 | 형 이름으로 쓰지 않음 | 현행 분류에서는 `표식` 또는 `탐지`로 재판정 |
| 복원형 | 형 이름으로 쓰지 않음 | `복원`은 효과 태그로 내려감 |
| 판독형 | 형 이름으로 쓰지 않음 | 보통 `탐지 + 판독 태그`로 읽음 |
| 각인학파(구 학내 속칭) | 보존각인학파 | 아카이빙 중심. 설명할 때는 `표식 + 탐지` 계열로 읽음 |
| 전장연산학파 | 군진선도학파 | 황탑 재편 후 현행 명칭 |
| 법정무효학파 | 서약집행학파 | 백탑 재편 후 현행 명칭 |
| 오염격리학파 | 단절추방학파 | 흑탑 재편 후 현행 명칭 |

### A.3 마법 분류 주석

- `각인`은 더 이상 정형화 마법의 형 이름이 아니다.
- 다만 `마나사인`, `연장각인`, `발췌각인`, `영상 각인`, `각인광장` 같은 기술명·서비스명·고유명사에는 남길 수 있다.
- 분류 설명에서는 항상 `표식` 또는 `탐지`로 다시 읽는다.
