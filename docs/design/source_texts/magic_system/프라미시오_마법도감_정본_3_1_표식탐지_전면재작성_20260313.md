# 프라미시오 마법 도감 3.1

## 부제

표식·탐지 개정을 반영해 `색 × 형 × 잔류` 문법을 다시 잠근 정본 - 웹툰 한 컷과 게임 한 버튼이 같은 문법을 쓰는 판본

\---

## 0\. 이번 정본에서 잠근 것

1. **정형화 마법의 전면 분류는 `색 × 형 × 잔류`다.**  
사용자는 먼저 색을 보고, 다음에 형을 보고, 마지막에 잔류를 본다. 태그는 필요할 때만 읽는다.
2. **형은 6형이며, 이번 판본의 6형은 `사출 / 전개 / 피복 / 기동 / 표식 / 탐지`다.**  
`복원`과 `판독`은 더 이상 기본 형이 아니다. 둘은 결과를 설명하는 **효과 태그**로 내려간다.  
`각인`도 더 이상 형 이름으로 쓰지 않는다. 이제 `각인`은 기술명·학파명·전통어로만 남고, 실제 분류에서는 `표식` 또는 `탐지`로 읽는다.
3. **잔류 축은 3분류만 전면에 둔다.**  
전면 표기는 `순간 / 유지 / 흔적`만 사용한다.  
`부착 / 설치 / 체류`는 `유지`가 **어디에 붙어 남는지**를 설명하는 2차 하위 태그다.
4. **모든 주문은 같은 표기법을 쓴다.**  
기본 표기는 다음과 같다.

```text
\\\\\\\[색 / 형 / 잔류 / 태그]
예) \\\\\\\[적 / 사출 / 순간 / 가열]
예) \\\\\\\[청 / 전개 / 유지 / 구속]
예) \\\\\\\[자 / 표식 / 유지 / 봉문]
예) \\\\\\\[자 / 탐지 / 흔적 / 재생]
```

5. **독자와 플레이어가 주문을 보고 묻는 질문은 셋뿐이다.**  
색: 이 주문은 무슨 힘인가?  
형: 이 주문은 화면에서 어떤 모양으로 보이는가?  
잔류: 이 주문은 어떻게 남는가?  
태그는 그 뒤에 읽는다. 태그는 회복, 정화, 추적, 구속, 반사, 침식처럼 **실제로 무슨 일이 나는지**를 설명한다.
6. **강한 주문은 반드시 읽히는 약점을 남긴다.**  
큰 화력은 큰 흔적을 남긴다.  
유지형은 반드시 지워질 수 있어야 한다.  
강한 표식형은 조건과 매개가 읽히면 약해진다.  
탐지형은 차폐와 잡음, 위장에 흔들린다.  
흔적형은 시간을 이기지 못한다.  
흑색과 문장법은 강하지만 공공 로그와 감독을 피할 수 없다.
7. **보존각인학파는 아카이빙의 학파로 재설계한다.**  
이 학파의 중심은 '기억'이 아니라 **정보를 표식으로 남기고, 탐지로 다시 꺼내 쓰는 일**이다.
8. **`작전연산학파`, `법정무효학파`, `오염격리학파`는 폐기한다.**  
황탑은 `군진선도학파`, 백탑은 `서약집행학파`, 흑탑은 `단절추방학파`로 다시 세운다.
9. **웹툰 한 컷과 게임 한 버튼은 같은 문법을 쓴다.**  
색 하나, 형 하나, 잔류 하나, 태그 최대 셋.  
이것이 이번 정본의 최우선 원칙이다.

## 1\. 이 도감이 다루는 범위

이 도감은 프라미시오에서 실제로 가르치고 전승하는 **정형화 마법**을 정리한 기술서다.  
따라서 `색 × 형 × 잔류`는 초월력 전체를 덮는 우주론이 아니라, **학파식·표준식·등록식·문장법을 읽고 가르치기 위한 실무 문법**이다.

프라미시오의 힘은 결국 시스템을 설득하는 여러 경로 위에 놓인다.  
그중 이 도감이 다루는 것은 다음 네 갈래다.

* 공리 마법
* 학파 기반 등록식
* 사실상 표준식으로 굳어진 공용 등록식
* 혈통 키를 필요로 하는 문장비전

심연 동기화, 매직 아키텍트, 워든 반응 같은 예외 현상은 정규 교본의 범위를 넘어서는 항목으로 분리한다.

## 2\. 마법 체계 4.1 - 색 × 형 × 잔류

### 2-1. 색

|색|핵심 힘|전투 인상|사회 기능|화면 인상|
|-|-|-|-|-|
|적|열압의 가산|폭발, 과열, 압박|제련, 공성, 화재 대응|열파동, 균열, 적광|
|청|구조의 가산|방벽, 고정, 구획|건축, 결계, 치안|격자, 앵커, 문턱|
|녹|생체·촉매의 가산|회복, 독, 생장|치료, 약리, 농업|섬유, 포자, 순환맥|
|황|벡터의 가산|가속, 넉백, 탄도|수송, 병참, 공성기|화살선, 궤적, 반동파|
|자|지각·보존의 가산|교신, 기만, 기록|전령, 외교, 서고|잔상, 초점 어긋남, 기록면|
|백|순도·서약의 가산|정화, 보호, 집행|의례, 치료, 성소|환형 성광, 백선, 정결면|
|흑|존재를 깎는 감산|침식, 맹점, 단절|위험물 해체, 금역 봉쇄, 추방|먹힌 공백, 잘린 선, 음영 결손|

### 2-2. 형

형은 주문이 **화면에 어떻게 보이는가**를 정리하는 축이다.

|형|설명|대표 화면|
|-|-|-|
|사출|쏘아 보내 맞힌다.|탄, 광선, 파편, 침|
|전개|면과 구역을 펼친다.|장판, 벽, 안개, 경계선|
|피복|대상에 둘러씌운다.|막, 외피, 갑주, 필터|
|기동|밀고 당기고 옮긴다.|돌진, 도약, 견인, 편이|
|표식|찍고 새기고 조건을 남긴다.|낙인, 인장, 룬선, 궤도선, 봉문선, 지시점|
|탐지|숨은 정보와 흔적을 드러내 읽는다.|주사선, 반응면, 윤곽 강조, 추적선, 유령 지도, 재생 프레임|

### 2-3. 잔류

잔류는 주문이 **어떻게 남는가**를 설명한다.

|잔류|정의|강점|약점|
|-|-|-|-|
|순간|한 번 터지고 끝난다.|반응이 빠르고 값이 싸다.|지속 압박이 약하다.|
|유지|몸, 장비, 지면, 공기 중 하나에 남아 버틴다.|장판, 버프, 함정, 여과막, 경계선에 강하다.|박리, 강풍, 정화, 좌표 교란, 상위 권한 해제에 흔들린다.|
|흔적|마법이 지나간 뒤 남은 기억층을 다시 읽거나 불러온다.|증거, 재생, 색인, 맵 기믹, 오의 설계에 강하다.|시간, 세정, 다른 마법 노이즈, 윤리 규제에 약하다.|

`유지`는 필요할 때만 다음 하위 태그를 붙인다.

* **유지·부착**: 사람, 장비, 문서, 물건에 붙어 남는다.
* **유지·설치**: 지면, 좌표, 문턱, 방, 구조물에 고정된다.
* **유지·체류**: 공기, 안개, 먼지, 빛, 소리, 그림자 같은 매질에 떠돈다.

전면 표기에서는 세부 태그를 숨기고 그냥 `유지`로 읽어도 된다.

### 2-4. 태그

태그는 한 주문이 실제로 하는 일을 읽히게 만드는 최종 정보다.

* 한 주문의 태그는 **최대 3개**까지만 붙인다.
* 태그는 먼저 보이는 뜻 하나가 분명해야 한다.
* 예: `가열`, `구속`, `회복`, `정화`, `추적`, `검증`, `반사`, `침식`, `추방`, `재생`

태그는 형을 대신하지 않는다.  
`회복`만 적힌 주문은 존재하지 않는다.  
`\\\\\\\[백 / 피복 / 유지·부착 / 회복, 보호]`처럼 읽혀야 한다.

### 2-5. 카운터 문법

강한 주문은 반드시 강한 카운터를 읽히게 해야 한다. 이번 정본은 이를 세계관 규칙으로 고정한다.

1. **큰 화력은 큰 흔적을 남긴다.**  
강한 적·황 주문은 반드시 열흔, 궤적흔, 반동흔 중 하나를 남겨 역추적의 실마리를 준다.
2. **유지형 주문은 반드시 지워질 수 있다.**  
유지형 주문은 다음 다섯 갈래 중 하나 이상에 흔들린다.  
`박리 / 정화 / 강풍 / 좌표 교란 / 상위 권한 해제`
3. **강한 표식형 주문은 조건이 읽히면 약해진다.**  
강한 표식일수록 트리거와 매개, 권한선이 더 뚜렷해진다.  
감춘 표식일수록 출력이 약하고, 강한 표식일수록 발동 조건이 선명하다.
4. **탐지형 주문은 차폐와 잡음을 이기지 못한다.**  
강한 탐지일수록 더 넓게 읽지만, 거짓 흔적과 다중 서명, 세정, 위장, 역탐지에 더 크게 흔들린다.  
완벽한 탐지는 금지하며, 항상 거짓 양성과 거짓 음성의 위험을 남긴다.
5. **흔적형 주문은 시간을 이기지 못한다.**  
오래된 흔적일수록 더 짧고, 더 엉키고, 더 비싸다.  
흔적을 되살릴수록 잡음과 윤리 문제가 커진다.
6. **흑색과 문장법은 공공 로그를 피할 수 없다.**  
흑색 상급식과 문장비전 상급 운용은 감독망에 자동 기록된다.  
그래서 이 힘들은 무법의 힘이 아니라, 관리 비용이 큰 힘이 된다.

### 2-6. 공리 마법 / 표준식 / 등록식 / 사실상 표준식 / 문장비전

* **공리 마법**: 시스템이 원래부터 허용한 공용 문법이다. 누구나 배울 수 있고, 가장 안정적이다.
* **표준식**: 공리 마법과 안정화된 등록식을 교육용 문법으로 묶은 것이다. 하급 교본은 대부분 이 층위를 다룬다.
* **등록식**: 창시자의 정신적 서명으로 세계에 등록된 술식이다. 학파는 이 등록식을 동조·전승하는 집단이다.
* **사실상 표준식**: 원래는 특정 학파의 등록식이었지만, 제도와 수요가 폭증하며 사실상 공용 기반기처럼 풀린 주문이다.
* **문장비전**: 혈통 키를 가져야만 쓸 수 있는 특수 등록식이다.

이번 정본에서 특히 중요한 것은 **사실상 표준식**이다.  
이 세계에는 시스템이 처음부터 준 공리 마법만 있는 것이 아니다.  
`마나사인`, `연장각인`처럼, 원래는 특정 학파의 등록식이었으나 너무 많이 쓰여서 사실상 공공 기반 기술처럼 굳은 주문들이 있다.  
이 이름들에 `각인`이 남아 있어도, 이번 정본에서는 전부 `표식형`으로 판정한다.

이 층위가 생기면서 보존각인학파는 몰락했다.
학파의 입문 등록식이던 것이 이제는 모든 마법사가 한 발씩 걸치는 공용 실무기가 되었기 때문이다.

### 2-7. 예외 현상과 금기

* **심연 동기화**는 논리 검증을 우회하는 예외이며, 학파식의 정점으로 취급하지 않는다.
* **매직 아키텍트**는 구축 기반의 특이 사례로서 일반 학파 전승과 분리한다.
* **워든 반응**은 현실 복구 기제에 가깝다. 강한 오의와 같은 척도로 재지 않는다.
* **생자 기억각인**과 **장기 흔적 인양**은 윤리 규제가 매우 강하다. 강한 흔적술일수록 감독과 보고 의무가 무겁다.



## 3\. 각 색별 표준식 입문서 재편

이번 정본의 입문서는 모두 같은 골격을 쓴다.

1. 주색 이론
2. 사고와 금기
3. 6형 입문식
4. 잔류와 보고서
5. 사회 응용
6. 기말 시연

흑색만 예외적으로 `공개 대응편 / 감독 실습편 / 봉문 부록`의 3부 구조를 유지한다.

### 3-1. 적색 표준식 입문 《열압》

* **배포 시점**: 하급 2학년 1학기
* **교육 목표**: 불을 크게 만드는 법보다, 열을 어디에 쌓고 언제 터뜨릴지 가르친다.
* **핵심 문법**: 적색은 `축열 → 표식 → 폭발`의 순서를 먼저 배운다.

**대표 입문식 6개**

* **화점탄** — `\\\\\\\[적 / 사출 / 순간 / 가열]` 작은 열핵을 직선으로 쏘아 약한 표면을 먼저 달군다.
* **온류장** — `\\\\\\\[적 / 전개 / 유지·체류 / 압박, 시야왜곡]` 좁은 구역의 공기를 덥혀 호흡과 조준을 흔든다.
* **열갑막** — `\\\\\\\[적 / 피복 / 유지·부착 / 강화, 가열]` 무기나 방패에 열 외피를 씌워 절단력과 압박을 올린다.
* **역분출보** — `\\\\\\\[적 / 기동 / 순간 / 도약, 넉백]` 반동 열분출로 짧은 후퇴와 측이동을 만든다.
* **발화표** — `\\\\\\\[적 / 표식 / 유지·부착 / 축열, 유폭]` 대상에 열 축적점을 찍어 후속 적색식의 효율을 올린다.
* **잔열탐지** — `\\\\\\\[적 / 탐지 / 흔적 / 판독, 역추적]` 최근 발화와 과열 경로를 잔류층에서 드러내 읽어낸다.

### 3-2. 청색 표준식 입문 《구조》

* **배포 시점**: 하급 2학년 1학기
* **교육 목표**: 막는 법보다 선을 긋고 하중을 나누는 법을 익히게 한다.
* **핵심 문법**: 청색은 탱킹 색이 아니라 `공간을 선으로 자르는 색`이다.

**대표 입문식 6개**

* **결속침** — `\\\\\\\[청 / 사출 / 순간 / 구속]` 맞은 지점의 움직임을 무겁게 만드는 기본 투사.
* **격자장** — `\\\\\\\[청 / 전개 / 유지·설치 / 구획, 차단]` 바닥과 벽에 얇은 구조 격자를 펼친다.

### 4-4. 하급 과정이 보존각인학파를 죽인 방식

보존각인학파는 예전에 `마나사인`, `연장각인` 같은 행정 실무로 돈을 벌었다. 이 주문들은 지금 기준으로 모두 `표식형` 공용 기반기다.  
문제는 이 기술들이 너무 유용해서, 제도와 시장이 그것들을 **보편 실무**로 바꾸어 버렸다는 점이다.

이제 학생들은 이렇게 생각한다.

* 마나사인? 누구나 한다.
* 연장각인? 접수처 실무도 한다.
* 진짜 비싼 보존술? 유지비가 너무 크고 윤리 규제가 무겁다.

그래서 보존각인학파는 **부전공 값은 높고, 주전공 값은 죽은 학파**가 되었다.

## 7\. 문장법 재정본 3.1

### 7-1. 수정 원칙

문장법도 이번 정본에서는 같은 문법으로 읽는다.  
혈통 키가 있어야 한다는 점만 다를 뿐, 독자에게는 여전히 `색 하나, 형 하나, 잔류 하나, 태그 최대 셋`으로 보여야 한다.

문장법 개정 원칙은 다섯 가지다.

1. **문장 객체가 먼저 보이게 한다.** 기둥, 문턱, 안개, 수면, 그림자, 깃발, 말뚝처럼 화면에 보이는 매개를 앞세운다.
2. **효과 본체를 인프라가 아니라 장면으로 읽히게 한다.** 결과는 행정과 통치에 걸쳐도, 장면은 `전개`, `표식`, `탐지`로 보이게 만든다.
3. **상급 문장법은 반드시 공공 로그를 남긴다.** 왕실과 귀족이라도 감독 비용에서 완전히 자유롭지 않다.
4. **강한 문장은 명확한 카운터를 가진다.** 기둥 파괴, 결절 절단, 상위 정본 제시, 강풍, 차광, 좌표 교란 같은 약점이 반드시 있다.
5. **문장비전도 흔적과 시간을 이기지 못한다.** 오래된 문장은 마모되고, 잡음이 끼고, 관리비가 급격히 오른다.

## 8\. 웹툰화·게임화 밸런스 메모

### 8-1. 웹툰 가독성

* 적색은 열 누적과 균열, 붉은 낙정으로 읽힌다.
* 청색은 선, 문턱, 격자, 앵커로 읽힌다.
* 녹색은 섬유, 포자, 생체막으로 읽힌다.
* 황색은 궤적, 화살선, 반동파, 깃선으로 읽힌다.
* 자색은 초점 어긋남, 잔상, 기록면, 재생 프레임으로 읽힌다.
* 백색은 환형 성광, 깨끗한 절단면, 순백 막으로 읽힌다.
* 흑색은 먹힌 공백, 사라진 선, 덜 존재하는 그림자로 읽힌다.

큰 주문일수록 시각적 계층이 더 선명해야 한다.  
특히 `표식`은 찍히고 남는 선과 인장으로, `탐지`는 훑고 드러내는 파면과 유령 지도로 즉시 구분되어야 한다.  
오의는 대부분 `공간 전체의 규칙이 한 장면으로 바뀌는 순간`으로 보이게 설계한다.

### 8-2. 게임 포지션

* 적탑: 원거리 폭딜 / 장비 파괴 / 시야 압박
* 청탑: 거점 유지 / 구속 / 지형 교정
* 녹탑: 전장 의무 / 성역 운용 / 환경 점유
* 황탑: 탄도 제어 / 금속·전기 / 분대 유도
* 자탑: 교신 / 허상 / 보존·증거
* 백탑: 생존 고정 / 광역 정화 / 서약 집행
* 흑탑: 침식 / 잠행 / 단절 추방

### 8-3. 파워 밸런스

1. **즉발 화력은 적·황이 강하다.**
2. **거점 유지와 정면 방어는 청·백이 강하다.**
3. **생존 관리와 환경 점유는 녹이 강하다.**
4. **장면 연출, 여론전, 증거전은 자가 강하다.**
5. **침식과 잠행, 추방은 흑이 강하지만 감독 비용이 가장 크다.**

### 9\. 결론

이번 정본의 핵심은 네 줄로 정리된다.

1. **정형화 마법은 `색 × 형 × 잔류`로 읽는다.**
2. **형은 `사출 / 전개 / 피복 / 기동 / 표식 / 탐지`의 6개이다.**
3. **잔류는 `순간 / 유지 / 흔적`만 전면에 둔다.**
4. **보존각인학파는 기억 조작이 아니라 아카이빙의 학파이며, 돈이 되는 주력기의 공용화 때문에 주전공 수요가 몰락했다.**

따라서 이 정본에서 독자와 플레이어가 읽어야 할 것은 언제나 같다.  
색 하나, 형 하나, 잔류 하나, 태그 최대 셋.  
그 뒤에야 세계관의 권력, 제도, 감독, 윤리가 따라온다.
