# Phase 0 Major Rename Map

## 원칙
- `메커니즘 설명어`보다 `장면/감각 환기어`를 우선한다.
- `7색`은 상위 감정/실루엣, `21학파`는 중층 전문화라는 독법을 유지한다.
- 이미 환기력이 충분한 이름은 억지로 바꾸지 않는다.
- `보존각인학파`는 학파명과 핵심 위상만 동결한다.

## 실행 메모
- 1차 리네임 확정 후보는 `12개`다.
- `review_later` 5학파는 1차 이름 시안이 충분히 상업적이면 유지하고, 그렇지 않으면 2차 후보로 승격한다.
- `keep`과 `locked`는 Phase 1 네이밍 검토 대상에서 제외한다.
