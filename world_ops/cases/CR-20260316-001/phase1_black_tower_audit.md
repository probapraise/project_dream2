## 1. 흑탑 1차 판정
- 흑탑의 상위 축은 이미 `침식 / 음영 / 공허`로 충분히 선명하다.
- 현재 흑탑의 공통 문법은 `결손`이고, 표면 이미지는 아래 셋으로 정리된다.
  - `서명과 외피가 깎여 나가는 침식 흔적`
  - `시선에서 밀려나는 그림자와 맹점`
  - `검은 천공과 반발, 비틀린 간섭선`
- 즉 흑탑의 현재 문제는 `탑 전체 방향`보다 `첫째 축의 주술 어감`과 `셋째 축의 옛 추방 문법`에 가깝다.
- 상태는 아래처럼 나뉜다.
  - `침식주술학파 -> 침식정련학파`: 이번 라운드에서 크게 손봐야 하는 리빌드 대상이다.
  - `음영잠행학파`: 학파명은 유지가 맞지만, 주문층은 `주시 감지 / 시선 빗김 / 암습 비수 / 사각 이동 / 그림자 조형·실체화 / 그림자 도약` 축으로 보정하는 편이 맞다.
  - `단절추방학파 -> 공허간섭학파`: 이번 라운드의 핵심 리빌드 대상이다.

## 2. 현재 상태 요약

### 2-1. 침식축
- 현행 대응: `침식주술학파`
- 1차 후보: `침식정련학파`
- 판정: `rename_now`
- 현재 강점:
  - 부식, 감산, 장기전이라는 흑탑 첫 축의 핵심 감각이 한 번에 읽힌다.
  - 위험물 해체, 유물 처리에 더해 `마도구 생산용 정련 축`으로도 확장 가능하다.
- 현재 약점:
  - `주술`이 야만 문명 샤먼, 몬스터, 주술계 문법을 먼저 떠올리게 해 현재 세계의 제도권 흑탑 공방축과 어긋난다.
  - 주문층 일부도 `저주`, `맹점`, `공허` 같은 옆 축 어휘가 섞여 축 경계가 흐리다.
- 판정:
  - 학파명과 주문층을 함께 정리하는 편이 맞다.

### 2-2. 음영축
- 현행 대응: `음영잠행학파`
- 판정: `keep`
- 현재 강점:
  - 음영, 맹점, 무음 이동, 침투가 즉시 읽힌다.
  - `보이지 않게 만든다`는 흑탑 두 번째 축의 임무가 매우 직관적이다.
- 현재 약점:
  - `맹점편이`, `주목삭감`, `경계문흔`, `빈자리위장`, `후열전이`, `흑야통로`는 이름이나 능력상 고점이 약하거나 자탑과 겹쳐 보일 여지가 있다.
  - 따라서 학파명은 유지하되 주문층은 `감지 / 시선 빗김 / 직접 암습 / 사각 이동 / 그림자 조형·실체화 / 그림자 도약 / 인과 역전` 문법으로 보정하는 편이 맞다.
- 판정:
  - 학파명 유지, 주문층 보정이 맞다.

### 2-3. 공허축
- 현행 대응: `단절추방학파`
- 1차 후보: `공허간섭학파`
- 현재 강점:
  - `공허`라는 표면 이미지는 흑탑 셋째 축으로 매우 강하다.
  - 소환, 방벽, 위험물, 대형 술식에 간섭한다는 기관 효용도 잘 붙는다.
- 현재 약점:
  - `단절추방` 문법은 행정적 차단과 격리 쪽으로 기울어, `모든 것을 거부하고 밀어내는 공허 에너지`를 직접 조작한다는 장면이 약하다.
  - 기존 12식도 `추방 / 귀속 / 문턱 삭제` 비중이 두꺼워, 천공과 파손, 간섭이라는 새 핵심 키워드를 충분히 못 살린다.
- 판정:
  - 학파명과 주문층을 함께 갈아엎는 편이 맞다.

## 3. 학파명 1차 판정
- `침식정련학파`
  - `침식`이 감산과 누적 붕괴를, `정련`이 마도구 생산과 마나석 서명 정제를 함께 품는다.
  - 흑탑 첫 축을 `검은 저주술`이 아니라 `불필요한 것을 깎아 핵만 남기는 제도권 감산 공방술`로 읽히게 만든다.
- `음영잠행학파`
  - 유지가 맞다.
  - `보이지 않게 만들고, 존재감을 주의 밖으로 밀어내고, 들키지 않은 채 통과한다`는 판타지를 가장 바로 보여 준다.
- `공허간섭학파`
  - `공허`가 만물을 거부하는 에너지의 표면 이미지를, `간섭`이 물질과 마력의 맞물림을 깨는 전투 문법을 함께 품는다.
  - 단순 추방술이나 차단 행정술이 아니라 `공허를 직접 꽂아 넣어 비틀고 뚫고 파손시키는 흑탑 셋째 축`으로 읽히게 만든다.

## 4. 흑탑 한 줄 결론
- 흑탑은 지금 기준으로 `탑 전체 방향`과 `3학파 12식 1차 점검`이 모두 닫혀 있고, 다음 실무는 소개문 압축과 live sync 후보 문장 추출이다.
- 따라서 실무 순서는 `흑탑 소개문 압축 -> live sync 후보 문장 추출 -> 승인 후 live SSOT 반영`이 가장 자연스럽다.

- 2) 흑탑 소개문에서 `침식 / 음영 / 공허` 3축 차이를 한 문단씩으로 줄인다.

## 7. 공허간섭학파 1차 판정
- 공허축은 흑탑의 `모든 것을 거부하고 밀어내는 공허 에너지를 직접 조작해 물질과 마력의 맞물림을 깨고, 구멍을 뚫고, 파손을 퍼뜨리는 손`을 담당한다.
- 핵심은 단순 추방이 아니라 `공허`, `간섭`, `천공`, `반발`, `파손`이다.
- 현재 저장소 기준 공허축은 `공허탄 -> 배격막/천공침 -> 간섭선/간섭판독 -> 반발파 -> 공허천공/파공탄 -> 간섭붕괴/배격장/공허파쇄 -> 공허붕락` 흐름으로 읽는 편이 맞다.
- 가장 중요한 보정은 아래 일곱 가지다.
  - 학파명은 `단절추방학파 -> 공허간섭학파`로 바꿔 추방 행정술이 아니라 공허 에너지 직접 조작 축으로 올린다.
  - `단절표 -> 공허탄`, `빈칸막 -> 배격막`으로 입문 장면을 `범용 공허탄 + 반발막` 문법으로 정리한다.
  - `절문침 -> 천공침`, `끊긴선 -> 간섭선`으로 초급 전투 문법을 `구멍 뚫기 + 간섭선`으로 다시 세운다.
  - `연결문양 -> 간섭판독`으로 바꿔 어디가 가장 잘 비틀리고 뚫리는지 읽는 공허식 판독 유틸을 확보한다.
  - `귀로삭제 -> 반발파`로 바꾸되, 강한 대신 근거리 한정이라는 조건을 명시해 직관성을 높인다.
  - `외연추방 -> 공허천공`, `공백수납 -> 파공탄`, `소환절단 -> 간섭붕괴`, `귀속박탈 -> 공허파쇄`로 바꿔 상황 대응형 공격기 패키지를 두껍게 만든다.
  - `절연외역 -> 공허붕락`으로 바꿔 외역형보다 단순무식한 최상위 직격 오의로 올린다.

## 8. 학파 한 줄 정의
- 공허간섭학파는 흑탑의 `공허 에너지를 직접 조작해 만물의 맞물림을 깨고, 구멍을 뚫고, 반발과 파손을 일으키는 손`을 담당하는 학파다.
- 핵심 판타지는 `공허탄`, `배격막`, `천공침`, `공허천공`, `파공탄`, `배격장`, `공허파쇄`, `공허붕락`이다.
- 기관 효용은 `방벽 파쇄`, `소환 붕괴`, `위험물 비활성화`, `공성 천공`, `대형 술식 교란`에서 나온다.

## 9. 리네임 채택 이유
- `단절추방학파`의 문제:
  - 기능은 맞지만 지나치게 설명형이고, 새로 정의한 `공허 에너지 직접 조작` 문법을 거의 보여 주지 못한다.
  - `단절`, `추방` 둘 다 결과어라서, 천공과 간섭, 반발이라는 표면 장면이 안 선다.
- `공허간섭학파`의 장점:
  - `공허`가 만물을 거부하는 에너지의 표면 이미지를, `간섭`이 물질과 마력의 결합을 깨는 전투 문법을 한 번에 압축한다.
  - 따라서 이 학파는 `연결 차단 행정술`이 아니라 `공허를 꽂아 넣어 비틀고 뚫고 파손시키는 흑탑 공허술`로 읽힌다.

| 중급 | 외연추방 | 공허천공 | 시그니처 | 응축한 공허를 한 점에 꽂아 갑주, 방벽, 벽체에 주먹 크기 이상의 관통공을 뚫는다 | 이 학파의 간판 천공식으로 충분히 강하다 |
| 중급 | 공백수납 | 파공탄 | 시그니처 | 응축한 공허탄을 쏘아 닿는 순간 넓은 파공 폭발을 일으켜 갑주, 벽체, 진형 일부를 한꺼번에 뜯어낸다 | 천공침과 공허천공 사이를 잇는 중급 광역 파괴기다 |
| 상급 | 소환절단 | 간섭붕괴 | 시그니처 | 적의 주문, 방벽, 소환, 장치에 공허 간섭을 직접 꽂아 넣어 구조를 들끓게 만들고 즉시 붕괴시킨다 | 사전 작업 없이도 대형 술식 교란과 붕괴가 가능하다 |
| 상급 | 문턱부재 | 배격장 | 시그니처 | 일정 구역을 공허 반발장으로 바꿔 들어오는 실체, 투사체, 주문을 밀어내고 억지로 통과하는 것엔 파손을 남긴다 | 공허축 대표 장판기로 매우 강하다 |
| 상급 | 귀속박탈 | 공허파쇄 | 시그니처 | 별도의 사전 작업 없이 응축 공허를 직격시켜 피격부 전체에 넓은 파손을 즉시 터뜨린다 | 단순무식하지만 강한 상급 직격기로 재정의하는 편이 맞다 |
| 오의 | 절연외역 | 공허붕락 | 궁극기 | 막대한 공허를 한 번에 떨어뜨려 넓은 범위의 실체와 주문, 방벽을 통째로 뒤틀고 꿰뚫고 붕괴시킨다 | 외역형보다 훨씬 직선적이고 강한 최상위 공격 오의다 |

## 11. 슬롯 균형 점검
- 현재 분포:
  - `기본탄 2`
  - `제어 2`
  - `방어·이동 1`
  - `시그니처 5`
  - `궁극기 1`
  - `비전투 유틸 1`
- 판정:
  - 권장 배치보다 `시그니처`가 더 두껍고, `비전투 유틸`은 얇다.
  - 그러나 이는 공허축이 본질적으로 `천공과 반발, 파손 확대`에 특화된 공격적 간섭축이라는 점과 잘 맞는다.
  - 오히려 이 편향 덕분에 음영축과 역할이 겹치지 않는다.
- 소개문용 대표 노출선:
  - `공허탄`
  - `천공침`
  - `공허천공`
  - `파공탄`
  - `배격장`
  - `공허붕락`

## 12. 기본 운용 방식 초안
- 공허축의 기본 루프는 `공허탄 압박 -> 배격막 진입 -> 천공침 개방 -> 간섭선 교란 -> 반발파 붕괴 -> 공허천공/파공탄 선택 -> 간섭붕괴/배격장/공허파쇄 -> 공허붕락`이다.
- 대표 순서는 아래처럼 읽힌다.
  - 먼저 `공허탄`이 범용 압박과 견제를 맡아 실체와 방벽, 주문에 가장 단순하게 공허를 꽂아 넣는다.
  - 급한 방어와 진입은 `배격막`이 맡아 접촉과 충격, 마력을 튕겨낸다.
  - 첫 개구부는 `천공침`이 뚫고, `간섭선`은 상대의 주문과 투사체, 감지선을 직접 무력화하거나 비틀어 흐름을 흐트러뜨린다.
  - `간섭판독`은 어떤 장치와 방벽, 주문이 가장 잘 비틀리고 무너질지를 읽어 준다.
  - 전장을 크게 흔들 때는 `반발파`가 근거리에서 대상과 주문을 한꺼번에 튕겨 균형을 무너뜨린다.
  - 본격 관통은 `공허천공`, 광역 뜯어내기는 `파공탄`이 맡아 상황에 맞춰 화력을 고른다.
  - 이어서 `간섭붕괴`, `배격장`, `공허파쇄`가 각각 술식 붕괴, 구역 배격, 단순 고위력 파손을 담당한다.
  - 최종적으로 `공허붕락`이 막대한 공허를 직격으로 떨어뜨려 넓은 범위를 통째로 붕괴시킨다.
- 즉 이 학파는 `대상을 행정적으로 추방하는 학파`가 아니라, `공허를 직접 꽂아 만물의 맞물림을 깨고 상황에 맞는 여러 공격기로 밀어붙이는 흑탑의 간섭술`이다.

## 15. 침식정련학파 1차 판정
- 침식축은 흑탑의 `불필요한 부분을 깎아 약하게 만들고, 마나석과 아티팩트에서 남길 핵과 서명만 남도록 정련하는 손`을 담당한다.
- 핵심은 샤먼식 저주가 아니라 `분해`, `박리`, `침식`, `감산`, `정련`이다.
- 현재 저장소 기준 침식축은 `분해선 -> 침식활주/부식흔 -> 침식장 -> 누적침식 -> 부식편 -> 층위분리 -> 서명소거/파손환원 -> 흑정련역` 흐름으로 읽는 편이 맞다.
- 가장 중요한 보정은 아래 열 가지다.
  - 학파명은 `침식주술학파 -> 침식정련학파`로 바꿔 제도권 흑탑 공방축 정체성을 앞세운다.
  - `식점 -> 분해선`으로 바꿔 입문식 첫 장면을 정밀 커팅으로 재정의한다.
  - `열화막 -> 박리막`으로 바꿔 침식축다운 방어식을 확보한다.
  - `맹점침 -> 침식활주`로 바꿔 침식을 활용한 기동식을 확보한다.
  - `저주문흔 -> 핵맥정렬`로 바꿔 침식을 이용한 회로 간편화 기반의 마도구 제작 전처리 유틸을 전면에 세운다.
  - `상쇄장부 -> 누적침식`으로 바꿔 누적 감산을 더 직관적으로 읽히게 만든다.
  - `공허편식 -> 부식편`으로 바꿔 타격마다 침식이 누적되는 이 학파의 주력 직접 공격기를 확보한다.
  - `서서히부재 -> 층위분리`로 바꿔 다층 구조와 복합 장비를 갈라내는 상급식을 준다.
  - `역류저주 -> 서명소거`로 바꾸되, `서명선별` 기능까지 통합해 단일 기능 정련을 맡긴다.
  - `재함몰 -> 파손환원`, `무흔식멸 -> 흑정련역`으로 올려 상급/오의의 장면 강도를 끌어올린다.

## 16. 학파 한 줄 정의
- 침식정련학파는 흑탑의 `갑주, 봉문, 구조물, 아티팩트, 마나석에서 불필요한 외피와 잡서명을 깎아 약하게 만들고, 필요하면 원하는 핵만 남도록 정련하는 손`을 담당하는 학파다.
- 핵심 판타지는 `분해선`, `부식흔`, `침식활주`, `누적침식`, `부식편`, `층위분리`, `흑정련역`이다.
- 기관 효용은 `마도구 생산`, `마나석 정련`, `위험물 해체`, `유물 분해`, `지속 감산전`에서 나온다.

## 17. 리네임 채택 이유
- `침식주술학파`의 문제:
  - `주술`이 원소술사, 샤먼, 몬스터 주술 같은 야만 문명 문법을 먼저 떠올리게 해 현재 세계의 제도권 학술원 흑탑과 어긋난다.
  - 마나석과 아티팩트 정련이라는 산업적 강점을 이름이 전혀 보여 주지 못한다.
- `침식정련학파`의 장점:
  - `침식`이 감산과 누적 붕괴를, `정련`이 마도구 생산과 단일 서명 추출을 함께 품는다.
  - 전투에서는 갑주와 보호층을 깎아 핵을 드러내고, 공방에서는 잡서명을 지워 필요한 기능만 남긴다는 양면성이 바로 읽힌다.
  - 따라서 이 학파는 `검은 저주술`이 아니라 `존재를 감산해 핵만 남기는 흑탑식 감산 정련술`로 읽힌다.

## 20. 기본 운용 방식 초안
- 침식축의 기본 루프는 `분해선 각인 -> 활주/부식 누적 -> 침식장 전개 -> 누적침식 증폭 -> 부식편 연타 -> 층위분리 -> 서명 소거/파손 환원 -> 흑정련역`이다.
- 대표 순서는 아래처럼 읽힌다.
  - 먼저 `분해선`과 `부식흔`으로 잘라야 할 선과 오래 갉아먹을 지점을 준비한다.
  - 급한 방어와 오염 차단은 `박리막`이 맡고, 기동은 `침식활주`가 진행선 위 마찰과 장애층을 깎아 돌파한다.
  - 마도구나 마나석 정련 전에는 `핵맥정렬`로 불필요한 갈래를 침식해 결맥을 간편화한 뒤 절삭과 각인 기준선을 먼저 곧게 잡는다. 다만 재료가 너무 많이 상해 회로 바탕이 무너지면 이 단계 자체가 성립하지 않는다.
  - 지속 압박은 `침식장`이 책임지고, 이미 쌓인 흔적은 `누적침식`이 한꺼번에 더 깊게 민다.
  - 한 줄 직격으로 핵심 장갑과 방호를 뚫어야 할 때는 `부식편`이 들어가고, 연속 타격이 누적될수록 같은 부위가 더 깊게 무너진다.
  - 복합 구조를 통째로 무너뜨리거나 부품을 분리해야 할 때는 `층위분리`, 기능을 정리해야 할 때는 `서명소거`가 들어간다.
  - 이미 수복된 장비, 구조물, 봉문, 아티팩트는 `파손환원`이 가장 깊은 파손 상태를 다시 끌어낸다.
  - 최종적으로 `흑정련역`이 구역 전체에서 `남길 것만 남기고 나머지를 깎아내는` 흑탑식 정련을 완성한다.
- 즉 이 학파는 `검은 저주를 거는 학파`가 아니라, `존재를 감산해 핵만 남기고, 그 과정 자체를 전투와 공방에서 써먹는 흑탑의 장기전 정련술`이다.

## 23. 음영잠행학파 1차 판정
- 음영축은 흑탑의 `보이는 윤곽과 들리는 흔적, 주시선과 존재의 바깥층을 깎아 감시망 사이로 스며드는 손`을 담당한다.
- 핵심은 직접 화력이나 공간 절단이 아니라 `은폐`, `무음`, `주시 감지`, `시선 빗김`, `사각 이동`, `그림자 조형`, `그림자 실체화`, `그림자 도약`, `인과 역전`이다.
- 현재 저장소 기준 음영축은 `주시 감지 -> 윤곽 억제 -> 시선 빗김 -> 그림자 비수 암습 -> 사각 이동 -> 무음화 -> 그림자 조형 -> 그림자 실체화 -> 암영도약 -> 현영전도` 흐름으로 읽는 편이 맞다.
- 가장 중요한 판정은 아래 여덟 가지다.
  - 학파명 `음영잠행학파`는 현행 유지가 맞다. 이름만으로 은폐와 잠입이 바로 읽힌다.
  - `맹점편이 -> 주시감흔`으로 바꿔, 지금 나를 보고 있는 시선부터 읽는 감지식을 입문에 올린다.
  - `주목삭감 -> 시선빗김`으로 바꿔, 존재감을 낮추는 효과를 `시선이 옆으로 미끄러진다`는 장면으로 다시 잡는다.
  - `경계문흔 -> 암영비수`로 바꿔, 근접전을 강제하지만 비인지 상태 찌르기에 크리티컬이 붙는 암습용 직접 공격기를 확보한다.
  - `그림자발 -> 사각보행`으로 바꿔, 전장에서 누군가의 시야가 닿지 않는 지점으로만 짧게 이동하는 중급 이동기로 재정의한다.
  - `소리박리 -> 무음박리`로 바꿔, 무음 운용을 더 직접적으로 읽히게 만든다.
  - `등불흔들기 -> 암영조형`, `빈자리위장 -> 암영실재`로 갈아, 그림자를 원하는 형태로 빚고 그 형태에 물리력을 부여하는 연계축을 세운다.
  - `후열전이 -> 암영도약`으로 올려 그림자에서 그림자로 뛰는 간판 이동기를 세우고, `흑야통로 -> 현영전도`로 갈아 범위 내 물체와 그림자의 인과 관계를 뒤집는 오의로 강화한다.

## 24. 학파 한 줄 정의
- 음영잠행학파는 흑탑의 `보이는 것과 들리는 것, 그리고 주시되는 정도를 깎아 그림자 사이로 스며드는 손`을 담당하는 학파다.
- 핵심 판타지는 `주시감흔`, `음영피복`, `암영비수`, `사각보행`, `암영조형`, `암영실재`, `암영도약`, `현영전도`다.
- 기관 효용은 `제한구역 경비`, `비밀 작전`, `감시 회피`, `침투`, `현장 은닉`에서 나온다.

## 25. 현행 유지 채택 이유
- `음영잠행학파`의 장점:
  - `음영`이 시야와 반사, 윤곽을 죽이는 표면 이미지를 잡고, `잠행`이 침투 목적을 즉시 보여 준다.
  - `도둑 / 암살자` 오해를 부를 수는 있지만, 실제 기능이 `은폐와 감시 회피`까지 포함한다는 점만 소개문에서 받쳐 주면 충분하다.
  - 흑탑 음영축은 뭔가를 직접 파괴하기보다 `보이는 것과 들리는 것, 그리고 존재의 바깥층을 덜어내는` 방향이라 현행 학파명이 잘 맞는다.
- 현재 약점:
  - `주시감흔`, `사각보행`은 기능을 잘 설명하지만 장면어 강도는 한 단계 더 밀 여지가 있다.
  - 다만 학파명 자체는 충분히 강하고, 이번 라운드에서는 주문층 보정이 우선이다.

## 28. 기본 운용 방식 초안
- 음영축의 기본 루프는 `주시 감지 -> 윤곽 억제 -> 시선 빗김 -> 비인지 암습 -> 사각 이동 -> 무음화 -> 그림자 조형 -> 그림자 실체화 -> 암영도약 -> 현영전도`이다.
- 대표 순서는 아래처럼 읽힌다.
  - 먼저 `주시감흔`이 지금 이 순간 자신을 보고 있는 시선을 그림자 위에 흔적으로 드러내 준다.
  - 기본 노출도는 `음영피복`이 낮추고, 정면으로 꽂히는 시선은 `시선빗김`이 옆으로 미끄러뜨린다.
  - 견제와 암습은 `암영비수`가 담당해 근접 찌르기를 강제하고, 비인지 상태에서 꽂히면 치명상이 크게 증폭된다.
  - 실제 이동은 `사각보행`과 `무음박리`가 맡아, 시야가 닿지 않는 짧은 거리만 골라 옮기고 소리는 벗겨낸다.
  - 끊긴 이동선과 암습 각은 `암영조형`이 그림자를 원하는 형태로 빚어 보완한다.
  - 그림자 자체를 전투 자산으로 끌어올릴 때는 `암영실재`가 조형된 그림자에 실제 물리력을 부여한다.
  - 핵심 돌파는 `암영도약`이 맡아 시선 여부와 무관하게 그림자에서 그림자로 바로 건너뛴다.
  - 현장 보관과 은닉은 `그림자은닉고`가 담당한다.
  - 최종적으로 `현영전도`가 구역 안의 그림자와 실체의 인과를 뒤집어, 그림자를 붙잡는 것만으로 실체를 제압하는 음영축 최상위 지배 장면을 만든다.
- 즉 이 학파는 `그림자 칼날로 죽이는 암살 학파`가 아니라, `보이는 것과 들리는 것, 그리고 그림자와 실체의 관계를 비틀어 감시망을 통과하고 전장을 장악하는 흑탑의 침투술`이다.

위 이름들은 `무엇이 덜 보이고`, `무엇이 덜 들리며`, `어떻게 감시를 비껴 가는가`가 대체로 한 번에 읽힌다.

### 재검토 여지 있는 이름
- `주시감흔`
  - 장점: 지금 나를 보고 있는 시선을 읽는다는 기능이 바로 보인다.
  - 문제: `감흔` 어휘가 약간 낯설 수 있다.
- `사각보행`
  - 장점: 비관측 지점 이동이라는 기능이 즉시 읽힌다.
  - 문제: 장면어보다 기능어 비중이 조금 높다.
