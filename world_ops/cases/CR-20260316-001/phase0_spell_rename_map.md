# Phase 0 Spell Rename Map

## 범위
- 아래 표는 1차 리네임 대상과 구조 재설계 대상 학파의 현행 주문명을 도감 정본에서 추출한 스켈레톤이다.
- `신명(초안)`은 Phase 1 검토가 끝난 블록부터 순차적으로 채운다. 아직 비어 있는 축과 후행 재검토 슬롯도 남아 있다.
- 모든 행에서 `괄호표기불변=true`다. 효과 태그, 색, 형, 잔류 표기는 건드리지 않는다.
