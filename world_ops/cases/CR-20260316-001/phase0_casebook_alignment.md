# Phase 0 Casebook Alignment

## 목적
- `fantasy_magic_casebook.xlsx`의 상업 문법을 Dream2의 불가침 구조 위로 번역한다.
- 이번 케이스는 `21학파를 폐기`하는 작업이 아니라, `21학파를 상업적으로 다시 읽히게 만드는 작업`이다.

## 번역 원칙

| 케이스북 권장 | Dream2 적용 방식 | 채택 여부 | 메모 |
|---|---|---|---|
| 학파 수는 4~6개가 안전 | `7색`을 독자 온보딩용 상위 카테고리로, `21학파`를 중층 전문화로 운용 | adapt | 21학파 자체는 유지한다. 독자는 먼저 7색, 그 다음 21학파를 이해하게 설계한다. |
| 학파마다 고정 슬롯을 만든다 | 각 학파 12전승을 `기본탄 / 제어 / 방어·이동 / 시그니처 / 궁극기 / 비전투 유틸` 축으로 재태깅 | adopt | 메커닉은 그대로 두고, 독자용 분류만 새로 만든다. |
| 저비용 시그니처 + 고비용 스펙터클을 병행한다 | 학파명/주문명/요약문에서 자주 쓰는 손맛 주문과 클라이맥스 주문을 명시적으로 대비시킨다 | adopt | 주인공/주요 라이벌 연출에도 그대로 연결 가능하다. |
| 비전투 마법을 반드시 포함한다 | 각 색과 학파의 생활/기관/인프라 효용을 따로 정리한다 | adopt | 드림2는 원래 실무/기관 마법이 강점이므로 전면화만 하면 된다. |
| 색·형태·사운드로 즉시 구분되게 설계한다 | 색별 감정/문화 레지스터와 실루엣 문법을 추가한다 | adopt | [WB-0029] 요약, 도감 정본 상세로 분담하는 편이 적절하다. |
| 상태이상과 콤보를 법칙으로 만든다 | 각 색의 핵심 상태이상과 제한된 교차색 시그니처 콤보를 canonize한다 | adopt | 전 조합표가 아니라 기억 가능한 소수 규칙만 채택한다. |
| 메커니즘보다 장면/감각이 먼저 읽혀야 한다 | 리네임 대상 학파/주문은 공학어가 아니라 장면 환기어로 재설계한다 | adopt | 괄호 표기와 기계 구조는 그대로 둔다. |
| 모든 마법이 다 강하면 안 된다 | 기존 `입문 -> 초급 -> 중급 -> 상급 -> 오의` 구조와 카운터 문법을 유지한다 | preserve | 파워 리듬은 이미 설계돼 있으므로 건드리지 않는다. |

## 이번 케이스에서 채택하지 않는 것
- `학파 수를 4~6개로 줄이는 구조 개편`
- `7색 × 6형 × 3잔류` 프레임워크 변경
- `보존각인학파` 학파명/핵심 위상 수정
- `문장비전`, `42개 색별 표준 입문식`, 커리큘럼 구조 수정
- 주인공 성장선에 맞춘 서사 허브 재설계

## 이번 케이스의 실제 산출물
- 학파 리네임 매핑표
- 주문 리네임 매핑표
- 색별 감정/문화 레지스터
- 독자용 슬롯 매트릭스
- 교차색 콤보 규칙
- alias 정책과 generated layer 재생성 계획

## 핵심 해석
- 이번 오버홀의 본질은 `설정 재창조`가 아니라 `표면 독법 교정`이다.
- 따라서 메커닉을 새로 짜는 대신, 이미 있는 12전승을 더 잘 읽히는 이름과 연출 슬롯으로 재배치해야 한다.
- 단, `보존각인학파`는 2026-03-17 작가 override로 예외 처리한다.
- 이 축은 학파명과 몰락/부흥의 핵심 서사는 유지하되, 전승 12식은 남길 것과 고칠 것을 나누어 부분 재점검한다.
