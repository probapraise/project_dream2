# Initial Design Frame

- change_id: CR-20260315-002
- status: draft

## 1. 재설계 순서
1. `표면 용어`를 다시 정한다.
   - 독자가 처음 듣는 이름
   - 작중 인물이 부르는 이름
   - 작가가 내부적으로만 부르는 이름
2. `기연 획득 장면`을 다시 정한다.
   - 주인공이 가장 먼저 손에 넣는 물건/계약/감각이 무엇인지
   - 그 순간 독자가 왜 “이건 먹혔다”고 느껴야 하는지
3. `성장 구조`를 다시 정한다.
   - 단계 수
   - 각 단계의 외형 변화
   - 전투/정치 효용
   - 대가와 약점
4. `커뮤니티 결박 구조`를 정한다.
   - 광장 성장 -> 기연 성장
   - 광장 위기 -> 기연 오염/약화
5. `전투/협상 응용`을 정한다.
   - 초반, 중반, 후반 장면에서 무엇을 읽고 무엇을 못 읽는가
6. `후행 세계관 조정`을 정한다.
   - 7색 체계
   - 21학파
   - 학술원 전공 구조

## 2. 현재 가장 중요한 선결 쟁점
- 쟁점 A. `LMM`을 작중 표면 명칭으로 유지할지, 작가용 약칭으로만 둘지
- 쟁점 B. `백지의 그리모어`를 초반 핵심 기연으로 확정할지
- 쟁점 C. 주인공의 특수 계약이 `픽시 계약`과 다른 무엇인지
- 쟁점 D. 각인광장이 완성되기 전에도 주인공 기연이 충분히 매력적으로 작동하는지
- 쟁점 E. 현재 `WB-0008`의 플랫폼 스펙을 얼마나 늦춰야 하는지
- 쟁점 F. `서고색인`과 `마나 사인 대조형 열람 제한`을 작중에서 하나의 마법처럼 보여주되, 작가용 구조는 어디까지 분리해서 관리할지

## 3. 현 시점 추천 기본안
- `LMM`은 작가용/후행 학술용 약칭으로만 두고, 표면은 `그리모어`, `정령 계약`, `광장`, `의도 읽기` 축으로 밀어붙인다.
- 초반 기연은 `색인의 그리모어` + `특수 정령 공명` 2축으로 시작하고, `백지의 그리모어`는 이후 성장형 별칭 후보로 남긴다.
- 현재 2단계 추천 경로는 `공공 색인 단말 -> 사적 기록 장부 -> 주석의 그리모어 -> 픽시 둥지화`다.
- 현재 3단계 추천 경로는 `주석 축적 -> 수동 연결 -> 실타래의 그리모어`다.
- 그리모어 자체의 기능 진화는 `실타래의 그리모어`에서 일단 닫고, `백지의 그리모어`는 이후 정령/성목 반응이 빈 페이지를 인터페이스로 쓰기 시작한 후반 상태명으로 재정의한다.
- 공용 인프라용 `픽시 계약`과 주인공의 특수 계약은 별개로 설계한다.
- 광장은 `기연의 원인`이 아니라 `기연의 성장 가속기`가 되어야 한다.
- `검색`은 초반 기능, `의도 읽기`는 중반부터 본체, `세계의 바닥 진동 감지`는 후반부 카드로 둔다.
- `서고/도서관`은 장서 대여소가 아니라 `문서 금고 + 열람권 기관`으로 확장 정의하고, 색인 아티팩트는 보안상 이유로 배치된 표준 단말로 설계한다.

## 4. 후속 라운드 제안
- 라운드 1: 표면 용어 + 기연 획득 구조 확정
- 라운드 2: 성장 단계표 확정
- 라운드 3: 전투/정치 응용표 확정
- 라운드 4: 7색 체계 / 21학파 / 학술원 전공 조정
