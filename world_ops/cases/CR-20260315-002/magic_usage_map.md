# Magic Usage Map

- change_id: CR-20260315-002
- status: confirmed
- focus: 커뮤니티 구현에 쓰이는 마법의 출처 — 기존 마법 vs 비표준 용법 vs 등록식 변질

## 1. 핵심 원칙

- 키리온이 쓰는 마법의 대부분은 **이미 세계에 존재하는 것**이다.
- 키리온의 특별함은 새 마법을 만드는 재능이 아니라, 기존 마법을 원래 용도와 다르게 쓰는 집요함이다.
- 명시적으로 등록식 변질이 일어나는 것은 **그리모어 진화(서고색인→주석→실타래) 딱 하나**다.
- 나머지 비표준 용법은 작중에서 당장 등록식으로 굳지 않는다. 키리온 본인도 의식하지 못한다.
- 다만 후반부에는 예외가 하나 생긴다.
  `현상 각인술`을 경량화한 `영상 각인술`은 변질이 아니라 의도적 연구와 교수 검증을 거친 `제도 등록식`이다.
- 등록식 생성이 쉬워 보이면 안 된다. 세계관 전체에서 개인 등록식을 만드는 것은 드문 일이다.

## 2. 세계에 이미 존재하는 마법 (학부 표준 커리큘럼)

### 2.1 기본 각인술
- 보존각인학파의 기초.
- 종이, 돌, 나무, 금속 등의 표면에 기록을 새기는 표준 기법.
- 학부생이 배우는 빵과 버터.

### 2.2 해독술
- 손상되거나 오래된 각인을 읽는 기술.
- 보존각인학파 필수 과목.
- 단계 2(보충 허용)에서 상한 기록 복원에 그대로 적용된다.

### 2.3 복원술
- 손상된 각인을 원래 상태로 되돌리는 기술.
- 해독술의 상위 과정. 학과 커리큘럼.
- 마찬가지로 단계 2에서 쓰이지만, 특별한 변형 없이 전공 실력으로 해결.

### 2.4 봉인술
- 기록을 보호하거나 열람을 제한하는 표준 보안 마법.
- 문서 관리, 기밀 유지, 위험 기록 격리에 쓰인다.
- 관청, 군, 학술원에서 널리 쓰이는 실용 마법.

### 2.5 픽시 계약
- 이미 확정된 공공 인프라 접속용 표준 프로토콜.
- 모든 학생이 배운다.
- 기록 전달, 배지 연동, 기본 중계에 사용.

### 2.6 마나 사인 인증
- 배지 시스템의 기반.
- 본인 확인, 권한 확인에 쓰는 표준 기법.
- 세계에 이미 존재하며, 키리온이 변형하지 않는다.

### 2.7 화상마법(투사 마법) 기초
- 배지에 이미 내장된 표준 기능.
- 저장된 정보를 있는 그대로 투사한다.
- 키리온이 깊이 다룰 필요 없다.
- `저장 형식 = 표시 형식` 원칙에 따라, 키리온은 저장 쪽만 신경 쓴다.

### 2.8 서고색인
- 그리모어의 원형 마법.
- 왕도급 서고에서 쓰는 고급 수제 아티팩트용 마법.
- 색인 원리 자체는 학과에서 이론으로 배울 수 있다.

### 2.9 현상 각인술
- 보존각인학파의 최상위 희소 기술.
- 시각, 청각뿐 아니라 냄새, 온기, 진동, 압력 변화 같은 복합 감각을 통째로 기록한다.
- 정보량과 유지 비용이 막대해, 실무에서는 거의 `의식용 / 증거용 / 연구용`으로만 다뤄진다.

### 2.10 고감각 환영 설비
- 학술원에 이미 존재하는 대형 체험형 환영 설비.
- 원래는 학생들의 모의 전투와 위험 상황 체험 교육에 쓰인다.
- 키리온은 이것을 새로 만들지 않고, 후반부에 `현상 각인술` 재생용으로 전용한다.

## 3. 키리온이 있는 그대로 쓰는 것

| 마법 | 용법 | 커뮤니티 단계 |
|------|------|-------------|
| 픽시 계약 | 기록 전달, 배지 연동 (표준 용법) | 전 단계 |
| 화상마법 | 배지 투사 (이미 내장, 키리온이 건드리지 않음) | 전 단계 |
| 마나 사인 인증 | 사용자 확인, 접근 권한 확인 (표준 용법) | 전 단계 |
| 해독술 | 상한 기록 읽기 (전공 수업대로) | 단계 2 |
| 복원술 | 손상된 기록 되돌리기 (전공 수업대로) | 단계 2 |
| 봉인술 (격리용) | 해로운 기록 격리 (원래 목적 그대로) | 단계 6 |
| 현상 각인술 | 복합 감각 기록 작성 (기술 자체는 기존 고급식) | 단계 8 |
| 고감각 환영 설비 | 현상 기록 재생 (기존 학술원 설비 전용) | 단계 8 |

- 특별한 변형 없음. 학과에서 배운 것을 배운 대로 쓰는 것.
- 다만 픽시 계약의 **사용 빈도**가 비정상적으로 높다는 점이
  픽시 둥지화를 낳고, 그리모어 진화의 물리적 메커니즘이 된다.

## 4. 키리온이 비틀어 쓰는 것 (기존 마법, 비표준 용도)

### 4.1 서고색인 → 현장 탐사 색인으로 전용
- **원래 용도**: 서고 장서 위치를 찾는다.
- **키리온의 용법**: 전장 기록, 탐사 분기, 기믹 위치를 찾는다.
- **성격**: 유일하게 등록식 변질로 가는 메인 라인.
  색인→주석→실타래 진화가 여기서 나온다. (Track A, 이미 설계 완료)
- **커뮤니티 단계**: 단계 0 전체

### 4.2 기본 각인술 → 성목 수심에 원격 각인
- **원래 용도**: 눈앞의 표면(종이, 돌, 금속)에 직접 새긴다.
- **키리온의 용법**: 기록을 인장 형태로 묶어 요정/픽시 편에 수심까지 보낸다.
  직접 수심 앞에 서서 새기는 게 아니라,
  그리모어에서 정리한 뒤 운반시키는 우회로.
- **성격**: 마법 자체가 바뀐 것이 아니라 **절차가 바뀐 것**.
  각인의 내용물을 미리 포장해서 보내는 것.
  어색하고 비효율적이지만, 요정의 협력 덕에 겨우 성립한다.
  새 마법이 아니라 `기존 각인술 + 요정 운반`의 조합.
- **커뮤니티 단계**: 단계 0 후반 ~ 전 단계

### 4.3 봉인술 → 권한 계층 부여로 전용
- **원래 용도**: 기록을 잠그거나 열람을 제한한다.
- **키리온의 용법**: 다른 학생의 배지에 제한된 상태 변경 권한을 부여한다.
  봉인술의 "이 사람은 열 수 있고 저 사람은 못 연다"는 권한 분기를
  역으로 이용해 "이 사람은 이것까지 할 수 있다"는 허가로 비트는 것.
- **성격**: 원래 `막는` 마법을 `조건부로 여는` 방향으로 비튼다.
  어색하고 우아하지 않다. 정식 권한 부여 체계가 아니라
  봉인의 조건 분기를 억지로 갈라 쓰는 것.
- **커뮤니티 단계**: 단계 6

### 4.4 기본 각인술 → 메타 색인 (기록 위에 색인 구조 각인)
- **원래 용도**: 빈 표면에 기록을 새긴다.
- **키리온의 용법**: 이미 기록이 새겨진 표면 위에 다시 색인 구조를 새긴다.
  나무에게 나이테 순서 말고 다른 방식으로 기억을 더듬는 법을 가르치는 것.
- **성격**: 원리는 같은 각인술이다. 다만 대상이 빈 표면이 아니라
  이미 기록이 있는 표면이라 간섭이 생기고 난이도가 훨씬 높다.
  새 마법을 발명한 게 아니라, 기존 각인술을 극한까지 밀어붙인 것.
- **커뮤니티 단계**: 단계 4

### 4.5 현상 각인술 → 환영 설비 재생으로 전용
- **원래 용도**: 복합 감각 현상을 통째로 박제해 연구나 증거로 남긴다.
- **키리온의 용법**: 학술원 기존 환영 설비와 연결해 `현상 열람실`에서 재생 가능한 기록으로 쓴다.
- **성격**: 새 마법을 만든 것이 아니라, 기존 희소 기술과 기존 대형 설비를 플랫폼 문법으로 묶은 것.
- **커뮤니티 단계**: 단계 8

## 5. 등록식 변질 현황

### 5.1 명시적 등록식 변질: 하나만
- **서고색인 → 주석 → 실타래** 진화가 유일한 등록식 변질 라인이다.
- 이것만으로도 작중에서 대단한 일이다.
- 등록식 생성 절차(`개인화 → 굳음 → 시스템 등록`)를 따르며,
  물리적 메커니즘은 픽시 둥지화다.

### 5.2 나머지 비표준 용법의 위치
- 원격 각인(4.2), 권한 전용(4.3), 메타 색인(4.4)은
  현 시점에서 등록식으로 굳었다고 보지 않는다.
- 키리온 본인도 이것들이 "새 마법"이라고 의식하지 않는다.
  기존 마법을 억지로 비틀어 쓰고 있을 뿐이다.
- 이 중 일부가 후행 서사에서 등록식으로 굳을 **가능성은 열려 있지만**,
  그것은 훨씬 뒤의 일이며 이 문서에서 확정하지 않는다.
- 중요한 것은: 독자가 "이 세계에서 등록식 만드는 게 쉽구나"라고 느끼면 안 된다.
  키리온의 비표준 용법은 어디까지나 불편하고 어색한 우회로로 보여야 한다.

### 5.3 예외: 의도적으로 제도 등록되는 `영상 각인술`
- 이것은 `그리모어`처럼 멋대로 진화한 등록식 변질이 아니다.
- 출발점은 기존 `현상 각인술`이다.
- 키리온은 상급 과정 연구 주제에서:
  - 현상을 감각별로 분해하고
  - 렌즈 촉매에 잡힌 시각 정보와
  - 마이크 촉매에 담긴 청각 정보만 남겨
  - 배지 재생 가능한 경량 기록으로 줄인다.
- 이 과정은 교수와 함께 실패 모드, 촉매 규격, 재현성, 교육 절차를 검증한다.
- 따라서 이것은 `개인 등록식 변질`이 아니라 `의도적 제도 등록식 개발`이다.
- 커뮤니티 단계로는 단계 9 `영상 각인 플랫폼`에 해당한다.

## 6. 커뮤니티 단계별 마법 사용 매핑

| 단계 | 핵심 동작 | 쓰이는 마법 | 출처 |
|------|----------|------------|------|
| 0 | 그리모어 진화, 요정 거래 | 서고색인(비표준), 기본 각인, 픽시 계약 | 비표준 + 표준 |
| 1 | 조회 전용 열람판 | 기본 각인(원격 인장), 픽시 계약, 마나 사인, 화상마법 | 비표준 + 표준 |
| 2 | 보충 허용, 상한 기록 수리 | 해독술, 복원술, 기본 각인 | 전부 표준 |
| 3 | 새 질문, 낡은 기록 정리 | 기본 각인, 봉인술(압축/봉인) | 전부 표준 |
| 4 | 검색 자유도 확대 | 기본 각인(메타 색인) | 비표준 |
| 5 | 추천/정렬 | 기본 각인(표식 연결) | 표준에 가까움 |
| 6 | 관리/정화 | 봉인술(격리=표준), 봉인술(권한 분기=비표준) | 표준 + 비표준 |
| 7 | 고밀도 기록 | 기본 각인(고난도 심층 각인) | 표준의 극한 |
| 8 | 현상 열람실 | 현상 각인술, 고감각 환영 설비 | 전부 기존 고급식/기존 설비 |
| 9 | 영상 각인 플랫폼 | 영상 각인술(신규 제도 등록식), 화상마법, 마나 사인 | 제도 등록식 + 표준 |
