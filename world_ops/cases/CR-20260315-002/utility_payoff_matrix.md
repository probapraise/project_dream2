# Utility Payoff Matrix

- change_id: CR-20260315-002
- status: draft
- focus: `전투 / 협상 / 정치 / 커뮤니티` 효용과 상한을 단계별로 고정한다

## 1. 문서 목적
- 설계 헌법이 요구한 `전투 / 협상 / 정치 / 커뮤니티` 재미를 한 표 안에서 관리한다.
- `강해진다`는 말을 전술 화력 하나로 축소하지 않고, 정보 구조가 주는 여러 종류의 보상으로 쪼갠다.
- 각 단계의 `상한`과 `금지선`을 같이 적어 과잉 파워를 막는다.

## 2. 효용 밴드

| 밴드 | 기준 단계 | 핵심 상태 |
|---|---|---|
| A | `색인` | 개인용 정보 우위 |
| B | `주석` | 정보에 의미를 붙이는 장인성 |
| C | `실타래` | 사건 단위 패턴 묶음 |
| D | `조회 전용 열람판` | 실습조 전술 우위 공유 |
| E | `보충/질문/검색` | 살아 있는 초기 커뮤니티 |
| F | `추천/정렬/정화/고밀도 기록` | 제도권 로컬 인프라 |
| G | `성숙한 LMM` | 서명 기반 의도 추정과 루트 활용 |
| H | `현상 열람실` | 희소한 복합 감각 증거와 고강도 체험 |
| I | `영상 각인 플랫폼 / LMM H~J` | 다중 업로더 시청각 플랫폼과 멀티모달 폭증 |
| J | `LMM K~L` | 루트 전용 금역 추론선 |

## 3. 효용 표

| 밴드 | 전투 효용 | 협상 효용 | 정치 효용 | 커뮤니티 효용 | 상한 / 금지선 |
|---|---|---|---|---|---|
| A | 길찾기, 함정 위치, 기존 기록 회수에서 압도적 우위 | 증거 문서를 빨리 찾아 작은 거래에서 우위 | 아직 정치력은 거의 없음. `희한한 학생`으로 보이는 수준 | 없음에 가깝다. 개인 도구 | 답이 없는 것을 만들어 내지 못함 |
| B | 원본 기록 옆에 현장 메모를 붙여 생존률 상승 | 같은 문서를 두고 서로 다른 해석을 제시하며 말빨이 세짐 | 교수/사서/선배 앞에서 `읽는 방식이 특이한 학생`으로 각인 | 아직 수작업. 공동체가 아니라 수기 장부 | 거짓말 탐지 아님 |
| C | 유사 함정, 몬스터 패턴, 보급 메모가 실타래로 묶여 전장 판단 속도 급상승 | 상대 주장과 유사 선례를 엮어 논리적 압박 가능 | 기숙사 내부 의존도가 생김. `정보를 가진 학생`이 됨 | 실습조 협력의 중심이 됨 | 정답 추천기 아님. 관련 기록 묶기까지만 |
| D | 전투 직전 공략 공유, 동일 기록 동시 확인, 약체 조의 생존력 상승 | 한 사람의 말이 아니라 `공개된 기록`을 근거로 협의 가능 | 누가 무엇을 볼 수 있는지가 곧 영향력. 엘프 클랜/교수의 시선이 붙음 | 처음으로 `우리 판`이 생긴다 | 학생은 쓰지 못하고 읽기만 가능 |
| E | 현장 보충과 새 질문이 쌓여 미지 대응 속도 상승 | 여러 조의 사례와 정정이 누적되어 협상 근거가 두꺼워짐 | 접근권, 공개권, 검색권을 둘러싼 미세 권력 형성 | 반복 참여자, 질문 문화, 규범의 싹이 생김 | 아직 대중 광장 아님. 운영자 의존 큼 |
| F | 추천/정렬, 정화, 고밀도 기록으로 전술망이 실제 인프라가 됨 | 누가 신뢰할 만한 기록자인지 판이 기억하기 시작해 설득 비용 감소 | 학파/기숙사/엘프 클랜/전심학파와의 이해관계 충돌이 본격화 | 로컬 광장 정체성 형성, 유지 비용과 오염 리스크도 함께 커짐 | 검열/편향/오판 위험 존재 |
| G | 전투 중 그리모어를 통한 양방향 질의, 역할 맞춤 기록 호출 | 상대의 `맥락`에 맞는 설명 패킷을 미리 준비할 수 있음 | 서명 패턴과 질문 흐름으로 집단 이해관계를 더 빨리 읽음 | 사용자는 `광장이 나를 이해한다`고 느끼기 시작함 | 감정 읽기, 독심술, 거짓말 탐지로 넘어가면 안 됨 |
| H | 현상 기록과 열람실 재생을 바탕으로 함정·오염·이상 징후를 가장 입체적으로 복기 | 말이 아니라 `같이 겪은 기록`이 협상 기준점이 되어 부정이 어려워짐 | 비리 폭로, 조사 증거, 교수 시연의 공개 등급이 새로운 권력 투쟁이 됨 | 플랫폼 체급이 커지지만 접근권과 유지 비용 갈등도 커짐 | 만능 증거 기계 아님. 기록 길이, 개수, 권역, 열람실이 제한됨 |
| I | 다수 학생이 올리는 시청각 영상으로 전술 학습 속도와 사례 누적이 폭증 | 상대에게 `보여 줄 수 있는 자료`가 늘어나 설득 방식 자체가 변함 | 보존각인학파 부흥, 업로더 서열, 영상 슬롯, 알고리즘 논쟁이 새 정치가 됨 | 광장이 스레드에서 미디어 플랫폼으로 진화하고 참여층이 폭증함 | 무제한 업로드, 무편집 진실, 완전 자동 검증으로 가면 안 됨 |
| J | 표면 전투보다 금역 대응, 실패 회피, 예외 존재 추적에 가까운 정보 우위 | 공개 협상용이 아니라 루트 전용 판단 재료 | `심연`과 `아키텍트` 추론은 공공 정치 자산이 아니라 봉인된 위험 지식이 됨 | 공용 커뮤니티 효용은 거의 없고, 오히려 숨겨야 유지된다 | 정답 예언, 안전한 승천 공식으로 다루면 안 됨 |

## 4. 영역별 손맛 포인트

### 4.1 전투
- 초반:
  - 남들이 모르는 길, 함정, 기믹을 먼저 잡는다
- 중반:
  - 여러 조의 시행착오를 하나의 전술 자산으로 묶는다
- 후반:
  - 전투 중 실시간 질의/응답과 역할 맞춤 정보 호출이 가능해진다

### 4.2 협상
- 초반:
  - 증거를 빨리 찾고 정리하는 능력
- 중반:
  - 누적된 사례와 질문 흐름을 근거로 상대를 설득
- 후반:
  - 상대가 실제로 원하는 기록 결을 더 잘 맞춰 제시

### 4.3 정치
- 초반:
  - `이상한 학생`에 대한 감시와 호기심
- 중반:
  - 접근권과 공개권이 권력이 됨
- 후반:
  - 엘프 클랜, 학파, 교수진, 기숙사 세력이 광장을 둘러싸고 이해관계를 만들기 시작함

### 4.4 커뮤니티
- 초반:
  - 정보 전달망의 씨앗
- 중반:
  - 여러 사람의 목소리와 질문이 쌓이는 살아 있는 기록고
- 후반:
  - 추천/정렬/정화/보존이 돌아가는 로컬 광장
- 최심부:
  - 현상 열람은 증거의 밀도를 올리고, 영상 플랫폼은 공공성을 폭발시키지만, 금역 추론선은 다시 비공개로 접힌다

## 5. 금지선
- 어떤 단계에서도 `상대의 현재 감정`을 읽어내는 기능으로 쓰지 않는다.
- `정치 효용`은 세뇌나 군중 조종이 아니라 `접근권`, `보존권`, `큐레이션`, `질문 흐름`의 우위에서 나온다.
- `협상 효용`은 독심술이 아니라 자료 구조와 맥락 맞춤 제시에서 나온다.
- `전투 효용`은 자동 승리 버튼이 아니라 정보 우위와 판단 속도에서 나온다.

## 6. 현재 추천 결론
- 이 기연의 재미는 전투에서만 터지면 부족하다.
- 반대로 협상/정치로만 가면 초반 손맛이 약하다.
- 따라서 초반은 `전투 우위`, 중반은 `협상/커뮤니티`, 후반은 `현상 열람 -> 영상 플랫폼 -> LMM`, 최심부는 `루트 전용 금역 추론`이 열리는 구조가 가장 자연스럽다.
