# CR-20260315-001 World Bible Restructure Plan

## 목적
- `프라미시오_무대용_설정_통합정리.docx`의 내용을 로노모고스 제외 기준으로 live world bible에 반영한다.
- 반영 우선순위는 `비공개 우주론 SSOT 확보 -> 표면 규칙 재정의 -> 구 메타 파편 제거 -> 인덱스/보조 문서 동기화`다.

## 설계 원칙
- 작중 세계의 공식 분류는 `표준식 / 등록식 / 문장비전`이다.
- `공리마법 / 구축마법`은 작가용 내부 설명으로만 유지한다.
- 우주론 코어는 전부 `META` 전용 문서가 직접 소유한다.
- 표면 문서는 가능한 한 `작중 인물들이 믿는 설명`을 기준으로 서술한다.
- 기존 `열구 / 외계 문명 유입 / 시뮬레이션 분기 세계` 축은 그대로 유지하지 않는다.

### 1. WB-0028 (신설)
- 제안 제목:
  - `Appendix G. 비공개 우주론 SSOT: 관찰자 / 심연 / 아키텍트 / 워든 / Axis / 앵커`
- 권장 visibility:
  - `META`
- 직접 소유할 내용:
  - 꿈의 덮어쓰기
  - 시스템의 설득과 검증
  - 위상 상쇄 간섭
  - 관찰자
  - 심연
  - 아키텍트
  - 워든
  - 심연 항해술
  - Axis / Branch
  - 앵커
  - 메아리 현상
- 비고:
  - 현행 우주론 파편을 여기로 회수하고, 표면 문서에서는 직접 설명을 최대한 제거한다.

### 2. WB-0029 (신설)
- 제안 제목:
  - `Appendix H. 적성 체계 공통 원리: 혈통 / 종족 / 권능의 서명 / 학파 / 마나색 / 마력석`
- 권장 visibility:
  - `PUBLIC` (본문 내부에 `CONFIDENTIAL` 표기 병행)
- 직접 소유할 내용:
  - 혈통 편중
  - 종족별 전문 분야 차이
  - 권능의 서명
  - 학파 = 서명 동기화 / 계승 집단
  - 마나색
  - 마력석의 존재 / 생성 / 활용
- 비고:
  - WB-0005, WB-0016, WB-0017, WB-0026이 이후 이 문서를 참조하도록 구조를 단순화한다.

## 재작성 우선 문서

### 1. WB-0005_magic_system.md
- 역할:
  - 표면 규칙의 중심 SSOT
- 재작성 방향:
  - `표준식 / 등록식 / 문장비전` 3분류를 공식 문법으로 고정
  - `7색 주색 체계`를 현대 실무 표준으로 재정의
  - 영창 / 마법진 / 촉매의 경험칙을 반영
  - 속성과 상성은 보조 프레임으로 낮춤
  - 신성 / 계약, 수련 초월은 각 경로 요약만 두고 심화는 별도 문서 참조

### 2. WB-0006_irminsul_infra.md
- 역할:
  - 특수 지형 노드 / 성목 / 정령군의 중심 설명 문서로 승격
- 재작성 방향:
  - 성목을 `특수 지형 노드`의 대표 사례로 일반화
  - 영험지, 신격성, 정령군, 잠든 신 개념 반영
  - 하급 정령이 인프라형 기능 존재라는 점을 표면 수준에서 정리

### 3. WB-0007_badge_network.md
- 역할:
  - `마나 사인 + 하급 정령 계약 + 배지` 실무 구조 설명
- 재작성 방향:
  - 배지를 `표준 마법 단말`로 유지하되, 활성 조건과 접속 구조를 새 기준에 맞춤
  - 하급 정령 계약 = 노드 인프라 접속권 구조 반영

### 4. WB-0016_social_stratification.md
- 역할:
  - 혈통 / 종족 / 차별 / 혼혈 취급의 표면 질감 정리
- 재작성 방향:
  - 종족별 전문화가 편견과 제도에 미치는 영향 강화
  - 벨쿠란의 관용적 외피와 인간 중심 봉건 구조를 동시에 반영

### 5. WB-0017_economy_resources.md
- 역할:
  - 마력석의 자원성 / 계승성 / 범죄 활용성을 경제 구조에 반영
- 재작성 방향:
  - 하급 몬스터산 마력석 인프라
  - 고급 마력석의 계승 보구화
  - 아티팩트형 / 동조 강화형 / 연료형의 시장 차이

### 6. WB-0026_appendix_crest_arcana.md
- 역할:
  - 문장비전과 혈통키의 특수성 정리
- 재작성 방향:
  - `확률처럼 보이지만 운명처럼 작동하는` 문장비전의 성격 강화
  - WB-0029 공통 정의와 충돌하지 않도록 문장비전 고유 축만 남김

## 교체/대체 우선 문서

### 1. WB-0010_planet_continents_naming.md
- 문제:
  - 행성 지리 설명 안에 구 우주론 파편이 삽입되어 있음
- 처리:
  - 대륙/국가/지리 정보는 유지하되, `세계 시스템 결함` 같은 직접 메타 표현 제거

### 2. WB-0011_echo_zone_rifts.md
- 문제:
  - 구 바닥 설정과 가장 직접적으로 결박된 문서
- 처리:
  - 현행 내용 유지 불가
  - 새 표면 설정이 확정되기 전까지는 `전면 재작성 대상`으로 분류
- 보류 쟁점:
  - `열구 / 외계 문명 유입`을 무엇으로 치환할지

### 3. WB-0013_timeline_event_engine.md
- 문제:
  - 최근 10년 연표의 출발점이 `마사토라 열구 재앙`
- 처리:
  - 초반 사건 엔진을 새 표면 갈등 기준으로 재배치

### 4. WB-0025_appendix_naming_constitution.md
- 문제:
  - `원본 지구 초기값 시뮬레이션` 서술이 직접 박혀 있음
- 처리:
  - 작명 규칙만 남기고 구 우주론 문장을 제거
