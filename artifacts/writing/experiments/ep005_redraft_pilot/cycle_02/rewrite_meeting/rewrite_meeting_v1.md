## 1. What still works

- `계승조회식 -> 미계승 -> 연회장 -> 세렌 첫 접속`이라는 회차 큰 골격
- 키리온만 별도 라인으로 불리는 검사 장면
- 결과 직후 `직계 쪽 분기 / 분기 다수의 무반응 / 데리온 어깨 이완` 흐름
- 세렌의 첫 앵커를 `너무 잘 보이는 자리에 혼자 있는 아이`로 두는 방향
- 리리아가 결국 세렌 쪽으로 먼저 움직여 관계를 여는 큰 기능

## 2. What does not work

- 연회장 이동 직전 정보 브리지가 약하다.
  - `아버지가 신세지고 있는 어르신`만으로는 곧바로 `라베르니온 공작가 별채`로 넘어갈 때 연결고리가 약하다.
  - 대충 읽는 독자는 `여기가 어디지`, `왜 왔지`를 잠깐 놓칠 수 있다.
- 연회장의 `1층 / 2층` 구조가 아직 장면 기능으로 선명하지 않다.
  - 1층은 아이들이 삼삼오오 모여 친교를 다지는 공개 쪽 공간이고
  - 2층은 각 가문이 자기 자리로 물러나 쉬는 반쯤 프라이빗한 공간인데
  - 지금은 구조 설명은 있으나 기능 대비가 몸으로 오지 않는다.
  - 특히 `리리아는 바로 1층으로 스며들고, 키리온은 곧장 2층으로 붙는다`는 성격 대비가 더 필요하다.
- 키리온의 관찰이 세렌으로 너무 빨리 이동한다.
  - 세렌으로 가기 전에
  - 다른 아이들이 어떤 무리를 만들고
  - 리리아가 그 안에서 어떻게 금방 섞이고
  - 계승/미계승, 중심/가장자리 차이가 어떻게 이미 생기는지
  가 더 깔려야 세렌의 고립이 더 이상하게 읽힌다.
- 연회장 대화가 작위적이고 접속 논리가 약하다.
  - `튀고 싶지 않은 거겠지 -> 그럼 더 답답하겠다`
  - `보기에 그래`
  - `그럼 우리가 가`
  - `남의 연회장에서 괜히 눈에 띄는 짓은 안 하는 게 좋아`
  같은 대사는 실제 대화 흐름보다 기능 전달이 앞선다.
  - `저 언니`에서 곧바로 `쟤`로 바뀌는 지칭도 흔들린다.

## 4. Keep / Cut / Add

### Keep

- 계승조회식 결과 직후의 온도 차
- 세렌이 처음엔 홀 중앙에 남아 있는 구도
- 리리아가 고립을 깨고 세렌을 끌어내는 기능
- 화말의 `저도 하고 싶어요.` 훅

### Cut

- `어르신`만 남겨 두는 모호한 브리지
- 1층과 2층 기능이 한 줄 설명으로만 지나가는 방식
- 세렌으로 너무 빨리 줌인하는 구성
- 기능 설명이 먼저 들리는 연회장 대사

### Add

- `라베르니온 공작가 별채에 왜 가는지`를 독자가 놓치지 않게 하는 짧고 명확한 연결
- 1층 공개 친교 공간과 2층 반프라이빗 가족 공간의 직접 대비
- 리리아는 1층에서 곧장 사람 속으로 들어가고, 키리온은 2층 난간으로 붙는 성격 대비
- 세렌 전 단계에서 `다른 애들이 어떻게 놀고 갈리고 있는지`를 키리온 시선으로 더 깔기
- 리리아/키리온 대사를 생활어와 실제 반응 중심으로 다시 쓰기

## 5. Desired read after rewrite

- 독자가 `왜 라베르니온 별채에 왔는지`를 멈춤 없이 이해한다.
- 1층과 2층의 성격 차이가 장면으로 바로 읽힌다.
- 리리아는 사람 속으로 섞이는 아이, 키리온은 떨어져 보는 아이로 더 선명해진다.
- 세렌의 고립은 홀 중앙 배치만이 아니라, 이미 형성된 다른 무리들과 대비되면서 더 쓸쓸하게 읽힌다.
- 연회장 대사는 `상황 설명`이 아니라 `실제 아이들 말싸움 직전의 호흡`처럼 들린다.

## 6. Hard no

- 라베르니온 별채 진입을 여전히 모호한 설명으로 넘기지 않는다.
- 1층과 2층을 다시 한 줄 구조 설명으로만 처리하지 않는다.
- 세렌을 다른 또래 무리와의 대비 없이 곧바로 특수한 아이로만 세우지 않는다.
- 연회장 대사를 기능 문장, 문어체 문장, 흔들리는 지칭으로 다시 쓰지 않는다.
