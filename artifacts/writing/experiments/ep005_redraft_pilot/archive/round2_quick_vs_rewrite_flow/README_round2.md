## What this archive is

- 이 보존본은 `사용자에게 바로 초고를 보여 주고`

## Why it was archived

- 사용자 피드백 기준, `user-facing draft` 앞에 항상 한 번의 내부 논리/담화 검수가 더 필요해졌다.
- 그래서 현재 파일럿의 기본 구조는 `raw -> logic review -> rewrite -> user-facing draft` 로 바뀌었다.
- 이 폴더는 그 이전 단계의 실험 기록만 남긴다.
