## Why this pilot is different

- 기존 초고도 비트 배치 자체는 이미 좋았다.
- 이번 파일의 차이는 `좋은 비트`를 더 추가한 게 아니라, 각 비트가 남겨야 하는 `독자 추론`과 `장면 증거`를 먼저 고정한 뒤 다시 썼다는 점이다.

## Notable shifts

- 대기실:
  - `분기 다수 / 직계 소수 / 렌바렌의 위장된 위치`를 명시 계약으로 먼저 못 박아 장면 증거가 더 선명해졌다.
- 검사 장면:
  - `들킬지도 모른다`가 아니라 `키리온은 그렇게 느낀다`에 맞춰 공포를 더 몸 반응 위주로 정렬했다.
- 연회장:
  - `계승/미계승 벽`을 설명보다 자리 배치와 접근 속도로 먼저 보이게 했다.
- 세렌:
  - 첫 인상을 `완벽함`보다 `중앙에 놓였지만 혼자인 아이`로 더 단단히 고정했다.
- 화말:
  - 게시판 구조 설명을 늘리지 않고, 세렌의 질문과 의지 표명으로만 닫았다.

## Decision use

- 이 파일군이 실제로 수동 리라이트 부담을 줄이는지 보려면, 다음 판단은 `문장 미감`보다 아래 항목으로 보면 된다.
  - 장면별 증명 의무가 더 분명한가
  - 수동 수정이 `구조 재수술`보다 `미세 조정`으로 줄었는가
  - 세렌/리리아/키리온의 역할 분담이 초고 단계에서 이미 읽히는가
