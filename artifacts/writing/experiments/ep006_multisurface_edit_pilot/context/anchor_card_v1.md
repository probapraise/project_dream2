# ep006 pilot anchor card v1

## 1. 작품 한 줄 정의

- `감시와 권한에 짓눌린 귀족가 차남이 정보 병목을 읽고 작은 구조 개입으로 살아남는 pre-academy 웹소설`

## 2. 장르 / 톤 / 문체 앵커

- 장르: 귀족가 생존형 성장 판타지, 정보 인프라 부트스트랩, 저강도 미스터리
- 톤: 건조한 긴장 + 생활 소음 + 낮은 농담 + 짧은 감정 균열
- 문체: 1인칭 근접 시점, 감각 -> 해석 -> 짧은 결론, 잦은 더블 엔터, 과한 문어체 금지

## 3. 주인공 앵커

- 이름: 키리온 렌바렌
- 현재 목표:
  - 들키지 않은 채 살아남기
  - 라베르니온 별채의 실무 혼선을 구조로 읽고 외부 환경에서도 통하는지 확인하기
- 결핍:
  - 가족 안에서조차 자기 위치가 불안정하다
  - `왜 나만 별도 검사 라인에 섰는가`를 아직 모른다
- 숨김:
  - 외래 기억의 출처
  - 과도하게 구조화된 사고 방식
- 말투 / 반응 습관:
  - 겉으론 여덟 살답게 낮추려 하지만, 속으로는 빠르게 분류하고 계산한다
  - 긴장은 손, 숨, 배 안쪽 수축, 시선 정리로 새는 편

## 4. 주요 인물 앵커

- 리리아
  - 역할: 실행 엔진, 현장 수집기, 분위기 점화기
  - 말투: 생활어, 손이 먼저 가는 즉답
  - 금지 오해: 설명 받아주는 마스코트로 납작해지면 안 된다
- 세렌
  - 역할: 높은 자리에서 고립된 아이, 외부 실무 증명의 첫 사용자
  - 현재 욕망: 아버지에게 칭찬받고 싶다
  - 금지 오해: 단순 `공작가 영애`, 즉시 친해지는 친구로 평면화하면 안 된다
- 칼리온
  - 역할: 감시자, 질문 대신 선택 구조를 여는 판정자
  - 금지 오해: 이미 주인공을 받아들인 보호자로 읽으면 안 된다
- 세르반
  - 역할: 효용을 즉시 계산하는 공작, 보상 제안자
  - 금지 오해: 자비로운 후견인처럼 먼저 읽히면 안 된다
- 데리온
  - 역할: 침묵과 긴장으로만 사건 무게를 드러내는 형
  - 금지 오해: 단순 강압형 형으로 고정하면 안 된다

## 5. 세계관 / 규칙 앵커

- 조기 마법 수련은 민감한 제도 리스크다
- 귀족 공간의 권한 문제는 `누가 해도 되는가`가 아니라 `누가 허락했는가`로 갈린다
- 라베르니온 별채는 렌바렌 저택보다 훨씬 정치적 시선이 촘촘하다
- 이번 화의 구조 개입은 혁명이 아니라 `저택 안 성공의 외부 번역`이어야 한다

## 6. 고정 복선 및 정보 공개 순서

- 별도 검사 라인 이유는 아직 풀지 않는다
- 수경원 / 방계 청소 / 칼리온의 최종 판정은 이번 화에서 드러내지 않는다
- 세렌의 고양감은 `아버지 인정 욕망 -> 유용한 중심` 순으로 체감돼야 한다
- 키리온의 장기 인프라 야망은 직접 선언하지 않고 현장 해결 충동으로만 드러낸다

## 7. 절대 건드리면 안 되는 것

- 키리온의 외래 기억 출처를 구체화하지 말 것
- 칼리온의 양자 제안 의도를 확정하지 말 것
- 세르반 / 3층 공간의 비밀 정보를 설명으로 풀지 말 것
- 세렌의 첫 접속을 로맨스나 즉시 친밀감으로 과장하지 말 것
- `외부 실무 증명`을 거대한 제도 혁신처럼 부풀리지 말 것

## 8. 흔들리기 쉬운 포인트 체크리스트

- 세렌의 훅이 다시 `익명 구조 호기심`으로 회귀하지 않았는가
- 구조 설명이 사람 욕망보다 앞서지 않았는가
- 3층 장면이 정보 dump로 변하지 않았는가
- 키리온의 거절이 감상적 연설이 되지 않았는가
- 마지막 문장이 다음 화 압박을 직접 남기는가
