## Usage rule

- 이 문서는 review 세션에서 승인된 변경만 모으는 SSOT다.
- 통합 반영 세션은 이 문서에 적힌 항목만 반영한다.
- review 단계에서 떠오른 아이디어라도 여기 없으면 통합 시 적용하지 않는다.

## Structure

- 현장 책임자 인정 뒤 `이 정도면 위에도 바로 말씀드려야겠습니다` 비트를 넣어 `1층 증명 -> 3층 이동` 브리지를 만든다.
- 3층 문 앞에서 하인이 `지금 손님이 계십니다`로 한 번 실제 제동을 걸고, 세렌이 익숙함으로 밀고 들어가게 한다.
- 세르반-칼리온 대화에 `연회 한 번에 하인 둘` 급의 경제 계산 비트를 넣어 양자 제안 전 압력을 모은다.

## Emotion / dialogue

- 세렌이 현장 인정 직후 위층 쪽을 한번 올려다보게 해 `아버지께 보여주고 싶다`는 욕망을 시선으로 먼저 보인다.
- 3층에서 세렌이 우리를 `흥분해서 끌고 온 것`보다 `증인 둘을 붙여 온 것`으로 읽히게 키리온 관찰을 한 줄 조정한다.
- 양자 제안 직후 리리아를 보는 순간이 키리온 거절의 직접 방아쇠가 되게 수정한다.
