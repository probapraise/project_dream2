## 1. 장면 맵

### 1) 세렌 요청 + 월권 제동
- 장면 기능: 세렌의 욕망과 키리온의 제동을 동시에 세운다.
- 진입 상태: `저도 만들고 싶어요.` 직후.
- 전환 계기: 눈앞에서 연회장 실무 혼선이 터진다.
- 종료 후 남는 질문: `지금 이 자리에서 허락받을 이유를 보여줄 수 있나?`

### 2) 석판 설치 + 현장 증명
- 장면 기능: 렌바렌식 게시판을 별채 연회장 환경에 맞게 번역한다.
- 진입 상태: 혼선은 보이지만 아직 공론화되진 않았다.
- 전환 계기: 세렌 권위 + 리리아 발품 + 키리온 설계가 합쳐져 하인들이 실제로 판을 쓰기 시작한다.
- 종료 후 남는 질문: `이게 정말 윗선까지 들고 갈 만큼의 결과인가?`

### 3) 현장 인정 + 3층 이동 브리지
- 장면 기능: `잘 굴러간다`를 `보고할 만하다`로 전환한다.
- 진입 상태: 하인들이 자발적으로 판을 붙잡기 시작한 상태.
- 전환 계기: 중간 실무 책임자가 효용을 인정하고, 세렌이 그 타이밍을 잡아 아버지에게 보여주겠다고 밀어붙인다.
- 종료 후 남는 질문: `세렌 권위가 3층 금선을 넘을 만큼 강한가?`

### 4) 3층 보상 협상
- 장면 기능: 실무 증명 -> 효용 계산 -> 위험한 보상 제안 -> 건조한 거절 -> 유예 보상으로 닫는다.
- 진입 상태: 키리온은 외부인이 오면 안 되는 자리라는 위화감만 느낀다.
- 전환 계기: 세르반의 `좋군`과 `뭘 받고 싶으냐`.
- 종료 후 남는 질문: `왕도 이틀은 보상인가, 관찰인가, 시험인가?`

## 2. 우선 수정 3개

1. `major` 1층 증명에서 3층 이동으로 넘어가는 보고 브리지가 필요하다.
   - 문제 구간: [`raw_draft_v1.txt:519`](/home/dlwhdgus/project_dream2/artifacts/writing/experiments/ep006_multisurface_edit_pilot/cycle_01/raw/raw_draft_v1.txt#L519)
   - 지금은 `중년 하인 인정 -> 세렌 즉시 3층 이동`으로 붙어 있어 기능은 맞지만 다소 급하다.
   - `위에 보고할 만하다`는 말 한 줄만 추가돼도 이동 이유가 훨씬 자연스러워진다.

2. `major` 3층 진입의 금선이 한 번 더 보여야 한다.
   - 문제 구간: [`raw_draft_v1.txt:627`](/home/dlwhdgus/project_dream2/artifacts/writing/experiments/ep006_multisurface_edit_pilot/cycle_01/raw/raw_draft_v1.txt#L627)
   - 하인이 비켜서는 장면은 이미 있지만, `지금 손님이 계십니다` 정도의 실질적 제동이 있으면 외부인 출입 불가 규칙이 더 단단해진다.

3. `major` 보상 협상은 효용 계산 뒤에 양자 제안이 더 건조하게 꽂혀야 한다.
   - 문제 구간: [`raw_draft_v1.txt:737`](/home/dlwhdgus/project_dream2/artifacts/writing/experiments/ep006_multisurface_edit_pilot/cycle_01/raw/raw_draft_v1.txt#L737)
   - 지금도 작동하지만, 세르반의 계산과 칼리온의 제안 사이에 `이게 왜 사람을 움직이는 보상까지 가는가`의 압력이 한 번 더 모이면 후반부가 더 매끈해진다.

## 3. 수정 원칙

- 1층 난장을 더 키우지 말고, 이미 보인 혼선이 `보고 가치가 있다`는 쪽으로만 눌러 준다.
- 3층은 초대장이 아니라 `세렌이 익숙함으로 밀고 들어가는 선 넘기`로 처리한다.
- 보상 장면은 감동이나 설명이 아니라 계산과 압력의 리듬으로 간다.
- 장면 재배열은 하지 않고, 기존 비트 사이에 짧은 브리지 비트만 삽입한다.

## 4. 패치안

### 패치 1
- 원문 일부:

```md
"덕분에 훨씬 낫습니다."
...
"아버지한테 보여드려야겠어요."
```

- 수정 방향:
  - 현장 책임자가 `이건 위에 보고할 만하다`고 먼저 말하게 해 3층 이동 이유를 만든다.

- 수정본:

```md
"덕분에 훨씬 낫습니다."

그 남자는 판을 다시 훑어보고 낮게 덧붙였다.

"이 정도면 위에도 바로 말씀드려야겠습니다."

세렌이 거의 튀어 오르듯 고개를 들었다.

"아니요. 제가 아버지께 보여드릴래요."
```

### 패치 2
- 원문 일부:

```md
"아가씨."

"아버지 계시죠?"

"예만……."
```

- 수정 방향:
  - 문지기 하인이 실제로 막고 싶어 하는 기류를 한 줄 더 드러낸다.

- 수정본:

```md
"아가씨."

"아버지 계시죠?"

"예, 다만…… 지금 손님이 계십니다."

그 말끝이 우리 셋에서 걸렸다.

"잠깐이면 돼요. 보여드릴 게 있어요."
```

### 패치 3
- 원문 일부:

```md
"효과는."

"반복 질문이 눈에 띄게 줄었습니다."
...
"그러면 양자는 어떻습니까."
```

- 수정 방향:
  - 세르반이 효용을 계산하는 비트를 조금 더 눌러, 양자 제안이 보상 후보로 튀어나오는 흐름을 자연스럽게 만든다.

- 수정본:

```md
"효과는."

"반복 질문이 눈에 띄게 줄었습니다."

"그 뒤로는 같은 대답을 여러 번 옮기는 발품이 줄었고, 하인 둘을 다른 일에 돌릴 수 있었지요."

세르반의 시선이 아주 짧게 내려앉았다.

"연회 한 번에 그 정도면 적지 않군."

그제야 칼리온이 다음 말을 꺼냈다.

"그러면 양자는 어떻습니까."
```
