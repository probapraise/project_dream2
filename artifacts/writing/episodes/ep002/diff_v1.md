# ep002 diff_v1

## 비교 대상
- 초안: `draft_v1.txt`
- 채택본: `2화 수정본.txt`
- canon 반영본: `canon/2화_리라이트_v1.md`

## 총평
- 수정본은 초안의 기본 비트 구조를 유지하면서, `키리온의 위장 논리`와 `정보 구조화 능력`을 더 전면으로 끌어냈다.
- 특히 집사실 장면이 단순 생활형 소동에서 `정보의 상태를 분류하는 사고`로 확장되어, 주인공의 강점이 더 선명해졌다.
- 마지막 칼리온 장면도 `칭찬 뒤 허가` 수준에서 끝나지 않고, 허가 자체가 더 정교한 감시 확장이라는 점을 더 분명하게 잠갔다.

## 핵심 차이

### 1. 오프닝의 무게중심 이동
- 초안은 `책을 덮은 선택이 맞았는가`에서 바로 시작한다.
- 수정본은 `검은 표지, 금박, 여덟 살이 읽기엔 졸린 책` 같은 감각과 위장 판단을 먼저 짧게 깔고 들어간다.
- 결과적으로 시작점이 더 차분하고, 키리온이 이미 계산 중이라는 인상이 강해졌다.

### 2. 리리아 대화의 기능 강화
- 초안의 리리아 장면은 `키리온다움`을 확인하는 데 집중한다.
- 수정본은 그 위에 `타인이 보는 키리온 데이터 수집`이라는 목적을 더 또렷하게 얹었다.
- `세상에서 날 제일 잘 아는 사람이 누구인지 시험해보고 싶어졌어` 같은 유도는, 키리온이 상대 자의식을 건드려 정보를 끌어내는 방식이라 더 영리하게 읽힌다.

### 3. 키리온 성향 정보의 구체화
- 초안은 `정리 안 된 것`, `반복 질문`, `서고 정리` 정도가 중심이다.
- 수정본은 여기에
  - 서랍을 열어놓고 안 닫는 것
  - 남이 이미 하는 일에 끼는 것
  - 장갑/꿀병/시럽병 같은 소규모 정리 전례
를 추가해, 원래 키리온의 성향을 더 입체적으로 만들었다.
- 덕분에 이후 안내판 행동이 더 자연스러워졌다.

### 4. 집사실 장면의 수준 상승
- 초안은 생활 병목을 보고 `확정/미정/누구에게 갈지` 정도로 정리하는 데 초점이 있다.
- 수정본은 질문을
  - 사실형
  - 시간 의존형
  - 유효기간이 있는 정보
로 읽어내고,
  - 확정
  - 미정
  - 소문
  - 다른 곳 먼저
라는 상태 기반 분류로 확장했다.
- 이 변화로 안내판이 단순 생활 팁이 아니라 `작은 정보 시스템`처럼 보인다.

### 5. 정보 이론의 자연스러운 삽입
- 초안에도 구조화는 있지만, 설명은 비교적 얕다.
- 수정본은
  - 답의 수명
  - 답의 소재지
  - 미정 자체도 정보
  - 답이 어디 있는지 모르면 반복 질문이 생긴다
를 자연스럽게 장면 안에 녹였다.
- 플랫폼 경영물 주인공다운 문제 인식이 훨씬 강해졌다.

### 6. 리리아의 역할이 더 선명해짐
- 초안에서 리리아는 귀엽고 빠른 조력자다.
- 수정본에서는
  - 사람들 사이에 들러붙는 감각
  - 필요한 말만 물고 오는 재주
  - 장면의 속도를 올리는 전달자
기능이 더 선명하다.
- 덕분에 리리아가 단순 마스코트가 아니라, 이후 커뮤니티 운영 조력자로 이어질 기반이 강해졌다.

### 7. 보상의 실무적 설득력 강화
- 초안은 집사의 `생각보다 쓸 만하군` 정도가 중심이다.
- 수정본은 실제 하인들이
  - 묻기 전에 답을 얻고
  - 다른 경로까지 한 번에 파악하고
  - 아예 집사실에 들어가지 않는 장면
을 더 구체적으로 밟는다.
- 그래서 작은 승리의 체감이 더 명확하다.

### 8. 칼리온 심문 파트의 질문 수준 변화
- 초안은 `왜 흑색을 안 읽었나`, `왜 저걸 했나` 중심이다.
- 수정본은 여기에 `어떤 기준으로 나눈 거야?`, `누가 가르쳐준 거야?`가 붙는다.
- 즉 행위가 아니라 `사고 방식의 출처`를 본다.
- 이 변화는 칼리온이 훨씬 더 위협적인 관찰자로 읽히게 만든다.

### 9. 엔딩의 압박이 더 명확해짐
- 초안의 마지막은 `보상이 아니라 다음 시험`이라는 감각을 잘 남긴다.
- 수정본은 그 뒤에
  - 서고에서 무엇을 읽는지
  - 어떤 순서로 읽는지
  - 어디서 멈추는지까지 전부 관찰 가능
하다는 후속 해석을 붙여, 왜 이 허가가 위험한지 명시적으로 강화했다.
- 마지막 `읽지 않은 것도, 기록되었다`는 문장은 초안보다 더 강한 잠금 역할을 한다.

## 채택본이 더 강해진 이유
- 키리온의 능동성이 단순 `생활 개선`이 아니라 `사고 방식 노출의 리스크를 감수한 정밀 위장`으로 읽힌다.
- 정보 구조화가 더 세밀해져, 훗날 게시판/플랫폼 설계로 이어질 개연성이 높아졌다.
- 리리아와 칼리온이 각각 `정서 자산`과 `관찰 압박`으로 더 분명하게 기능한다.

## 남는 체크 포인트
- 수정본은 초안보다 분석 밀도가 높아졌다. 이후 리비전에서는 설명이 아니라 장면으로 충분히 소화되는지 한 번 더 보면 좋다.
- `자색` 표기와 현재 live 용어 `자` 체계의 정합성은 이후 집필본 통일 시 다시 한 번 점검할 필요가 있다.
