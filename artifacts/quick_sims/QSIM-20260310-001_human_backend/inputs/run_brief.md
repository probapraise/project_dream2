# Run Brief

## Board
- `BOARD-001` 낙서장
- tone: 혼돈, 냉소, 내부자 유머
- anonymity_mode: 완전 익명
- moderation_style: 주인공 직접 관리

## Seed Event
- 누군가 낙서장에 `최상위 자동 응답식`이라 광고하는 새 각인 서비스 링크를 올렸다.
- 질문 1회당 열람권 1개가 든다.
- 열람권을 벌려면 먼저 `인간 응답 탭`에서 남의 질문에 답해야 한다.
- 어떤 학생이 사역마 이름 추천 요청에 `1번 2번 3번`이라고 답해 열람권 2개를 벌었다.
- 그걸로 `대현자 언제 옴?`을 물었더니 `곧 나와 동기들이 갈 거야. 넌 아직 준비가 안 됐다.`라는 답이 왔다.

## Intended Trigger
- 허접한 인터페이스 + 인간 백엔드 노역 구조 + 의미심장한 허접 답변이 동시에 붙을 때
- 낙서장식 발상의 전환 농담, 성능 은어 roleplay, 허접함 승인 문법이 터지는지 확인

## Guardrails
- META 금지
- live 문서 수정 금지
- explore only
- 정식 board state 승격 금지
