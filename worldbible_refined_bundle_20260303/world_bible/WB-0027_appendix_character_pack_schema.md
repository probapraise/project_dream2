### G.1 Character Card Schema (v1.0)
각 캐릭터 카드는 (1) 오프라인 캐릭터 조형(클리셰/욕망/결함), (2) 온라인 페르소나(보드 선택/말투/트리거), (3) 정합성 근거(왜 그 보드를 쓰는가)를 최소 단위로 묶는다.

| 필드 | 타입 | 설명 |
| --- | --- | --- |
| id | string | 불변 ID(CHAR-*, FST-* 등). |
| name | string | 표시명(작중 이름). |
| species | string | 종족(인간/엘프/드워프/리자드포크 등). |
| origin | string | 출신(국가/가문/조직). |
| role | string | 아카데미 내 역할(학년/직책/외부 파견 등). |
| tower | string | 칠탑(주색) 및 핵심 계열. |
| dorm | string | 기숙사(일반동/귀족동/장학생동/연구·탑동/외국·인외동 등). |
| trope_tags | list[string] | 국내/일본 아카데미물의 왕도 태그(전학생/학생회/라이벌/영애 등). |
| offline_core | string | 오프라인 1문장 코어(욕망/성향을 요약). |
| desire / fear / flaw | string | 욕망/공포/결함(서사 엔진용). |
| secret | string | [PUBLIC]/[CONFIDENTIAL]/[META] 레벨로 표기하는 비밀/정보 비대칭. |
| relationships | list[string] | 핵심 관계 2~4개(동맹/대립/이용). |
| boards | list[board_entry] | 주 활동 보드 3개+부 활동 0~2개. 각 보드별 페르소나/톤/핸들 스타일 포함. |
| avoids | list[string] | 회피하는 보드/금기(노출 위험). |
| psych_rationale | string | 보드 선택/페르소나가 심리적으로 타당한 이유. |
| voice | string | 말투/문장 길이/호칭/밈 사용 정도(보이스 가이드). |
