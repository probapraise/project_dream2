# Community Grammar — Layer B

## 정의

레이어 B는 **작가 투사 레이어**다.
주인공(빙의 개발자)이 전생에서 사랑했던 인터넷 커뮤니티 문법을 각인광장 안에 의도적으로 이식하는 계층.

실제 커뮤니티의 글·사건을 분해 → ATOM으로 저장 → 일정량 누적 시 상위 GRAMMAR 규칙으로 합성하는 파이프라인으로 운영한다.

---

## 적용 범위

### 결정 (2026-03-04)

레이어 B는 **각인광장 내 주인공이 직접 관리하는 단일 익명 게시판 "낙서장"**에만 적용한다.

#### 성격

| 항목 | 내용 |
|---|---|
| 소유/관리 | 주인공이 가장 아끼고 직접 운영 |
| 목적 | 순수 취미 — 주인공 개인 취향에 의해서만 유지 |
| 상업성 | 없음. 각인광장의 상업적 성장·LMM 고도화 등 외적 목표와 **의도적으로 배치** |
| 익명성 | 익명 게시판 |
| 서사적 기능 | 주인공의 고집과 정체성을 드러내는 핵심 장치 |

#### 내러티브 마찰

- 작품이 진행될수록 주인공 주변의 조력자들은 이 게시판의 존재를 문제시한다.
- "제발 저 문제 많은 익명 게시판 좀 없애고 쉽게 좀 가자"는 하소연이 반복된다.
- 그럼에도 주인공은 포기하지 못한다 — 이 공간은 주인공이 전생에서 그리워하는 커뮤니티 경험의 재현이기 때문.
- 이 갈등은 주인공의 캐릭터 아크에서 **합리적 경영자 vs. 감성적 커뮤니티 덕후** 사이의 긴장으로 기능한다.

#### 적용 범위가 단일 게시판인 이유

- 레이어 B의 문법은 특유의 익명성·혼돈·내부자 유머에 기반하므로, 각인광장 전체에 깔면 톤이 오염된다.
- 국소적 적용이 서사적 마찰(조력자들의 반대)을 자연스럽게 만든다.
- 주인공의 집착 대상이 명확해야 감정선이 선명하다.

---

## 파이프라인

```
입력: 작가가 실제 커뮤니티 글 + 설명 제공
  ↓
분해: 3레이어 추출
  - 사회 동학 (core_pattern)
  - 감정 레지스터 (emotional_register)
  - 커뮤니티 문법 (in_group_mechanics)
  ↓
저장: ATOM-NNN (YAML 형식, 본 문서 하단)
  ↓
합성: 10~15개 누적 시 → 상위 GRAMMAR-* 규칙으로 통합
  ↓
반영: 캐릭터 빅5/파생지표 조건과 연결 → 시뮬레이션 발동 조건
```

---

## ATOM 저장 형식

```yaml
ATOM-NNN:
  source_type: 게시글 / 댓글 / 밈 패턴 / 사건
  core_pattern: 추상 사회 동학
  emotional_register: 감정 온도 (냉소적 쾌감 / 훈훈한 연대 / 집단 조롱 등)
  in_group_mechanics: 집단 내 작동 원리
  trigger_condition: 발동 조건 (캐릭터 타입 + 상황)
  world_translation: 각인광장 표면형 (세계관 언어 번역)
```

## ATOM 저장소

```yaml
ATOM-001:
  source_type: 게시글 (스크린샷 + 한 줄 캡션)
  core_pattern: >
    역전된 의존 관계 — 강자(시스템)가 고장나서 약자(사용자)에게 도움을 구하는 상황이 발생했을 때,
    사용자가 분노 대신 "나를 필요로 해주잖아"라는 애착으로 반응한다.
    망가진 것에서 가치를 발견하는 시선.
  emotional_register: >
    온기 있는 냉소 — 표면은 조롱("ㅈㄴ웃기네 ㅅㅂㅋㅋ")이지만 대상을 파괴하려는 의도 없음.
    웃기니까 좋고, 좋으니까 공유한다는 단순한 쾌감 회로.
  in_group_mechanics: >
    - 맥락 설명 없이 던지기: 아는 사람만 웃으면 됨
    - 한 줄 반응 경쟁: 짧을수록 위트 있게 취급
    - 변주 확산: 같은 웃음 포인트를 각자 다른 각도로 비틂 ("AI역전세계", "ASI시대 미리보기")
    - 닉네임 무위계: ㅇㅇ(IP), 댕댕충, 볼통통 — 권위 없는 평등 참여
  trigger_condition: >
    높은 개방성 + 기술 리터러시가 있는 캐릭터가
    시스템 오작동/예상 외 결과를 목격했을 때,
    보고(리포트) 대신 놀이로 전환하는 성향
  world_translation: >
    각인광장 버전: 각인 시스템(마법 인터페이스)이 오작동하여
    사용자에게 역으로 질문을 던지기 시작함.
    어떤 유저가 그 대화를 스크린샷 찍어 게시판에 올리며
    "이 각인, 나를 필요로 하는 거 아닌가?" 한 줄 첨부.
    댓글란은 한 줄 반응과 세계관 드립으로 도배.
```
